-= BaguetteBox Drupal module =-

Summary
=========================
This module provides integration with BaguetteBox library.

Requirements
=========================
 * BaguetteBox should be installed to sites/all/libraries/baguettebox directory.

Installation
=========================
 * Install as usual, see http://drupal.org/node/895232 for further information.
 * Download BaguetteBox library from https://github.com/feimosi/baguetteBox.js
 * Unzip the library and place files from dist directory to sites/all/libraries/baguettebox directory.

Notes
=========================
 * You can use `drush baguettebox-download` command for easy installation of the library.


## Installation

After enabling this this module as normal you need to:

1. Add an admin API Token from HipChat and the default room to receive
   messages at /admin/config/services/hipchat
2. Download the hipchat-php library from Github
  https://github.com/hipchat/hipchat-php  and place it inside the hipchat module
  directory.
  If you placed the hipchat module at:
    * sites/all/modules/contrib/hipchat
  The plance the library at:
    * sites/all/modules/contrib/hipchat/hipchat-php
  So you have your HipChat.php at:
    * sites/all/modules/contrib/hipchat/hipchat-php/src/HipChat/HipChat.php

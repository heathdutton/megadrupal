Webform Constant Contact Drupal Module

---------------------------------------------------
REQUIREMENTS
---------------------------------------------------

Before installing the module please ensure you have both the Constant Contact and Webform modules installed and working.

---------------------------------------------------
INSTALLATION
---------------------------------------------------

To install the module upload all files to your /sites/all/modules/ folder.

Login to your drupal admin panel and browse to the manage modules page (/admin/modules), check the checkbox next to the "Webform Constant Contact" module and the click the save configuration button, the module is under the "Webform" section.

The module is now enabled and you can use the webform component while building forms.

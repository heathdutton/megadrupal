FFmpeg Video Thumbnail
======================
A simple file (field) formatter that use FFMpeg to extract a single frame of a
video and display it using core's image formatter or the picture module
(http://drupal.org/project/picture).

Installation
------------

Follow the standard module installation guide (http://drupal.org/documentation/install/modules-themes/modules-7)
to install FFmpeg Video Thumbnail. The module depends on the PHP FFmpeg library
integration module (http://drupal.org/project/php_ffmpeg), so follow the
instructions on its project page to install the third-party libraries that this
module requires.

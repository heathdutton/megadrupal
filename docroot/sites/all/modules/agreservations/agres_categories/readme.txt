AGRES_CATEGORIES README

This module depends on
agreservations,agres_bookings,duration,ls_titles and ls_block_titles

It installs the Contenttype Name:Unit type category, Type:agres_category
The fields in this Contenttype contain a configuration that will influence
the behavior of agreservations. Example Categies can be: Room, House, Car, Boat.
In the Blocks section for each node of type agres_category there will be a block
that can be enabled. This block will contain a form similar to those
provided by the agres_onlineform modules.

The purpose of this module is to make agreservations more flexible and dynamic.

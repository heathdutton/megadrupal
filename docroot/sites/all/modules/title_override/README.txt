Module: Title Override
Author: Bryan Burkholder <http://drupal.org/user/703590>
Author: Marcin Pajdzik <http://drupal.org/user/160555>

Description
===========
This module leverages the Context module to set or override page titles per path or 
other context conditions. The module provides a context reaction and supports tokens.

Installation
============
* Copy the 'title_override' module directory in to your Drupal
sites/all/modules directory as usual.

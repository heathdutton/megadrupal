Panels curator
==================

This module works with Drupal 7.x

Panels curator provides content administration tools for panels panes buckets. This allows you to group panes into buckets and remove items from those buckets to limit the panes that are available.


Installation
------------

1. Put the module in 'sites/all/modules/contrib'

2. Enable the module on the 'admin/build/modules' page.


OPENID PROVIDER AX
------------------

This module provides an API for exposing user related information as OpenID
fields according to the OpenID AX specification
http://openid.net/specs/openid-attribute-exchange-1_0.html

Typically, OpenID Provider AX is used with together with fields of the user
entity or the fields attached to a profile2 [1] entity to expose user profile
information as OpenID AX fields.


[1] http://drupal.org/project/profile2

Boost support for Aegir
=======================

This Drush module will alter the Aegir site vhost configurations to add the
right rewrite rules. It will also create the boost directories needed for 
proper operation, with proper permissions.

Installation instructions
-------------------------

Drop this directory in Aegir's Drush path (e.g. /var/aegir/.drush).

Enable the boost module on your site, then run Verify from Aegir.

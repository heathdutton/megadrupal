Main SMS
http://drupal.org/project/sms_mainsms

SUMMARY:
sms_mainsms is a drupal module to add the main_sms.ru gateway to the sms framework module.

REQUIREMENTS:
1. https://drupal.org/project/smsframework

DEMO:

INSTALLATION:
Install as usual, see http://drupal.org/documentation/install/modules-themes/modules-7 for further information.

UPGRADING:
No action necessary when upgrading from Drupal 6 to 7.

CONFIGURATION:
1. Visit SMS Framework configuration page at /admin/smsframework/gateways;
2. Select Main SMS as default;
3. Fill API fields as http://mainsms.ru/office/api_account filled.

-- SUMMARY --

This module enables a extensible free Share Button ecosystem for the
best possible social sharing optimization of your website.

For a full description of the module, visit the project page:
  http://drupal.org/project/social_share_counter

To submit bug reports and feature suggestions, or to track changes:
  http://drupal.org/project/issues/social_share_counter

REQUIREMENTS
============
- This module is written for Drupal 7.x.
- The 7.x branch of jQuery Update module (jquery_update).


INSTALLATION
============

1. Download the module from Drupal.org/project/social_share_counter and save it to your
   modules folder.
2. Enable the Social Share Counter module at admin/build/modules.
3. Set the access Social Share Counter permissions for authenticated users at
   admin/user/permissions.

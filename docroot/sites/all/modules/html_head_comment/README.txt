HTML Head Comment
-----------------

The simple module adds a html comment in the html page head with for example information about the site producer.


Example
-------

This is an example of how the module add a comment in the section of the html-source:

<!--

Site made by

 _                       _                     
| |                     (_)                    
| |__   __ _ _ __  _ __  _ _ __   ___  ___ ___ 
| '_ \ / _` | '_ \| '_ \| | '_ \ / _ \/ __/ __|
| | | | (_| | |_) | |_) | | | | |  __/\__ \__ \
|_| |_|\__,_| .__/| .__/|_|_| |_|\___||___/___/
            | |   | |                          
            |_|   |_|                          

http://www.happiness.se

-->


Ascii art optimized settings
----------------------------

In the settings the textarea for the message is rendered with monospace font so it's possible to make ASCII art. A nice ascii text generator can be found here:

http://www.network-science.de/ascii/


There are settings for:

  * Activating Site made by module. 
  * Entering the message

AdvAgg Search & Replace
-----------------------
Tap into the aggregated code generated by the excellent Advanced CSS/JS
Aggregator (advagg) [1] to replace strings in the output with alternatives, akin
to a preprocessor on the aggregated code.


Dependencies
------------------------------------------------------------------------------
This module requires the Advanced CSS/JS Aggregator module, it will not run if
it's not installed.


Configuration
------------------------------------------------------------------------------
admin/config/development/performance/advagg/replace


Usage
------------------------------------------------------------------------------
admin/config/user-interface/advagg-replacements

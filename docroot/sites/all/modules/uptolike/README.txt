RUSSIAN VERSION
---------------

ОПИСАНИЕ
--------
Интеграция социальных кнопок от uptolike.ru.

ВОЗМОЖНОСТИ
-----------
- Создание пресетов с настройками социальных кнопок.
- До 10 блоков социальных кнопок.
- Вставка кнопок в любое место у сущности (ноды, таксономии, пользователей).
- Фильтр контента: [uptolike] и [uptolike:preset] будут заменены на
  социальные кнопки.
- Просмотр статистики кнопок прямо на сайте.

ЗАВИСИМОСТИ
-----------
Chaos tool suite (ctools) https://www.drupal.org/project/ctools

УСТАНОВКА И НАСТРОЙКА
---------------------
1. Скачайте этот модуль и распакуйте его в папку sites/all/modules/contrib.
   Смотрите https://www.drupal.org/documentation/install/modules-themes для
   подробностей.
2. Включите этот модуль.
3. На странице admin/config/system/uptolike отредактируйте или добавьте новый
   пресет с настройками кнопок.
4. На странице admin/config/system/uptolike/settings выберите количество
   блоков социальных кнопок и отображение у сущностей.
5. На странице управления блоков выберите "Uptolike share buttons block 1" и
   кликните "настоить". Укажите нужный пресет настроек и выберите регион
   отображения блока. При необходимости проделайте тоже самое для других блоков.
6. На странице "Управление отображением" (например admin/structure/types/manage/
   article/display) включите поле "Uptolike" и укажите престет с настройками.

СПОНСИРОВАННО
------------
Разработка этого модуля спонсирована Uptolike.ru http://uptolike.ru.

ENGLISH VERSION
---------------

INTRODUCTION
------------
Integration of uptolike.ru social sharing buttons widget.

FEATURES
--------
- Create presets with the settings of the social buttons.
- Up to 10 blocks of the social share buttons.
- Insert the buttons to any place of the entity (nodes, taxonomy, users).
- Content filter: [uptolike] and [uptolike:preset] will be replaced with
  share buttons.
- View share buttons statistics directly on the website.

DEPENDENCIES
------------
Chaos tool suite (ctools) https://www.drupal.org/project/ctools

INSTALLATION AND CONFIGURATION
------------------------------
1. Download this module and unpack into sites/all/modules/contrib. See
   https://www.drupal.org/documentation/install/modules-themes for details.
2. Enable this module.
3. On the page admin/config/system/uptolike edit or add a new preset with
   the social share buttons settings.
4. On the page admin/config/system/uptolike/settings select the number of blocks
   of the social buttons and display on the entities.
5. On the block manage page, select block "Uptolike share buttons block 1" and
   click "settings". Select the desired preset and select the region
   to display. If necessary do the same for other blocks.
6. On the "Manage display" (for example admin/structure/types/manage/article/
   display) enable field "Uptolike and specify preset.

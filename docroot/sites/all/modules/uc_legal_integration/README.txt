-- SUMMARY --

The Ubercart Legal integration module creates a new ubercart checkout pane to
integrate the options provided by the Legal module into the Ubercart checkout
process.

For a full description of the module, visit the project page:
  http://drupal.org/project/uc_legal_integration

To submit bug reports and feature suggestions, or to track changes:
  http://drupal.org/project/issues/uc_legal_integration

-- REQUIREMENTS --

Legal module
Ubercart module

-- INSTALLATION --

Install as usual, see http://drupal.org/node/70151 for further information.

-- CONFIGURATION --

Enable the new created pane in admin/store/settings/checkout
Note: If there's no Terms and Conditions text set, the pane will not show up,
you can do it in legal module settings page (admin/config/people/legal).

Toolbar Visibility module allow to remove the
Drupal 7 Toolbar on selected theme(s).
This module is great for remove the Toolbar on
the front site and keep a separate admin theme.
For use this module you must disable Overlay Module.

Installation
------------

1. Copy toolbar_visibility folder to modules directory.
2. At admin/build/modules disable the Overlay module.
2. At admin/build/modules enable the Toolbar Visibility module.
4. Configure the module at admin/config/user-interface/toolbar-visibility

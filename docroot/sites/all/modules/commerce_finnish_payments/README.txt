
Drupal Commerce Finnish payments

Installation:

  1. Grab the latest version of FpiAPI from https://github.com/tcmug/fpiapi
     and install it to your libraries folder.
  2. Enable module
  3. Do the necessary configurations in admin/commerce/config/payment-methods

Information required for testing services can be found in the FpiAPI
documentation.

Payment methods are preconfigured to use testing credential information. You
can override settings by editing payment method reaction settings.

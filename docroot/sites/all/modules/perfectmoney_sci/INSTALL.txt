-------------
Requirements:
-------------
 - Drupal 7.x
 - Path

-------------
Installation:
-------------

Just upload perfectmoney_sci folder to modules directory of your Drupal 
installation and enable Perfect Money SCI in Modules section of admin 
interface. No extra actions are required, everything are installed automatically.

-------------
Configuration:
-------------

1) Go to Administer -> Perfect Money SCI -> Currencies and enter account(s) you want	
   to accept payments to. Also don't forget to enable desired currencies.

2) Enter Alternate PassPhrase under Settings section.

3) Go to Sample settings section and copy-paste setting to        
   https://perfectmoney.is/settings.html (ATM you just need to enter Alternate PassPhrase).

4) Go to Permissions section and set up permissions for the module.

These simple steps are actually all you need to do to start accepting Perfect Money 
payments via this URL:
http://yoursite.com/perfectmoney/payment

If you need more advanced information about things like module API & hooks, 
consult to README.txt

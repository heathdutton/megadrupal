
-- SUMMARY --

The Makemeeting module provides a field to allow users to provide their
availability on some dates choosen by the content author.

For a full description of the module, visit the project page:
  http://drupal.org/project/makemeeting

To submit bug reports and feature suggestions, or to track changes:
  http://drupal.org/project/issues/makemeeting


-- REQUIREMENTS --

None.


-- INSTALLATION --

* Install as usual, see http://drupal.org/node/70151 for further information.


-- CONFIGURATION --

* Configure user permissions in Administration » People » Permissions:

  - Provide an answer to makemeeting forms

    You might want to give this to authenticated users as it is required to
    provide an answer to a makemeeting form.

  - Delete any makemeeting answers

    This is intended for administrators and moderators to delete answers made
    by anonymous users in case of mistakes.


-- USAGE --

* Add the field associated with its widget to any content type


-- ROADMAP --

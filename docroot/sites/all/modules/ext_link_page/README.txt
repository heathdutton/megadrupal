--- INSTALLATION --------------------------------------------------------

1. Place the ext_link_page folder in your modules directory.

2. Enable "External Link Page" under Administer > Modules.

3. To adjust settings, go to: Administer > Configuration > User interface >
   External link page.

4. To add the external link page filter for text content, go to: Administer >
   Configuration > Content authoring > Text formats.

5. Ensure permissions are set. The "Access the external link page" permission is
   needed for users to access the exit page.

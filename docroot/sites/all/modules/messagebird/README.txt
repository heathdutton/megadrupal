Description
----------

MessageBird is an API which allows integration with MessageBird and a
Drupal site to send text messages.

Installation
------------

 * MessageBird depends on the libraries module, download and install it
   from http://drupal.org/project/libraries
 * Download PHP Class https://github.com/messagebird/php-rest-api
   and place the files in sites/all/libraries/messagebird
 * Enter authentication information
   http://yoursite.com/admin/config/services/messagebird

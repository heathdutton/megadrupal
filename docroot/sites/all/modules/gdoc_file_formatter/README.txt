This module implements file formatter that will display files using
embedded Google doc viewer.

Google viewer supports various types of files (.pdf, .doc, .docx,
.xls, .xlsx, .ppt, ...). 

See http://support.google.com/docs/bin/answer.py?hl=en&answer=1738646 
for more info.

You must enable File entity (http://drupal.org/project/file_entity) module
in order to use this formatter.

If you're looking for a field formatter check Embedded Google Docs Viewer
(http://drupal.org/project/gdoc_field).

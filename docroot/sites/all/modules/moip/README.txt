-- RESUMO --

Este módulo integra o MoIP (https://site.moip.com.br/) ao Drupal Commerce. Com uma API de alto nível e uma interface fácil de usar, este módulo possibilita uma solução "fora da caixa" para pagamentos com e-commerce no Brasil, e permite customizações com facilidade, quando necessárias.

O principal objetivo deste módulo é ter uma base inicial sólida, com uma abordagem orientada a objetos que permite a mudança de funcionalidades facilmente, sem a necessidade de hackear o código do módulo, permitindo a concentração dos esforços na lógica de negócios do e-commerce e processos de cada projeto.

Para uma descrição completa do módulo, visite a página do projeto:
  https://drupal.org/project/moip

Para submeter bugs e sugestões de funcionalidades, ou para acompanhar mudanças: 
  https://drupal.org/project/issues/moip


-- REQUERIMENTOS --

Este módulo possuiu 2 dependências:

  - Moip PHP Library - https://github.com/moiplabs/moip-php
  - Moip JS Library - https://github.com/moiplabs/moipjs

Ambas bibliotecas devem ser instaladas em sites/all/libraries, nos seguintes sub-diretórios:

  - /moip-php-master
  - /moipjs

(A biblioteca moipjs precisa apenas do arquivo moip.min.js (encontrado em /build) para funcionar)




*** ENGLISH ***


-- SUMMARY --

Integrates Drupal Commerce and MoIP, a payment services provider from Brazil, with a high level API and an easy to use interface, enabling an out-of-box solution for payments with e-commerce in Brazil and allowing customizations with ease when needed.

The main goal with this module is to have a solid starting point, with an Object Oriented approach that let's you change behaviors easily without need to hack anything, enabling us to concentrate efforts on the e-commerce business logic and processes of each project.

For a full description of the module, visit the project page:
  https://drupal.org/project/moip

To submit bug reports and feature suggestions, or to track changes:
  https://drupal.org/project/issues/moip


-- REQUIREMENTS --

This module has 2 dependencies:

  - Moip PHP Library - https://github.com/moiplabs/moip-php
  - Moip JS Library - https://github.com/moiplabs/moipjs

Both libraries should be installed in sites/all/libraries, in the following sub-directories:

  - /moip-php-master
  - /moipjs

(The library moipjs needs only the file moip.min.js (found in /build) in order to work)

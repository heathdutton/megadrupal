Two Level Comment, as its name tells, is a small module that allows you to keep
exactly two levels of comment. It could be used for Question/Answer style nodes.

== Configuration

1. Enable the module.
2. Edit a content type.
3. Under the "Comment settings" section enable "Threading" option, then enable
   "TLcomment mode" option.

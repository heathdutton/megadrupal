Blocks

The contents for All Classified Ads blocks can be themed using either a core template/theme function, as a default, or a specific alternative allowing you to target the specific instance of that core template. You can also override the complete block with the normal block-*.tpl.php templates.

Alternatives take the same parameters as the default.

Popular

  • item-list is used by default
  • item-list--classified-popular.tpl.php is the specific alternative

Recent

  • item-list is used by default
  • item-list--classified-recent.tpl.php is the specific alternative

Stats

  • table is used by default
  • table--classified-stats.tpl.php is the specific alternative

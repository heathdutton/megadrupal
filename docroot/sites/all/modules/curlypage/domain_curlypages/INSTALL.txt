Domain Curlypages installation instructions:
1. Enable Domain Curlypages
2. Navigate to Admin->Structure->Curlypages and assign curlypages to domains (by default all curlypages show up on all domains)

Domain Curlypages uses domain access grants similar to those in Domain Access module (domain_site and domain_id).
A user must have set domain access permission to assign curlypages to domains.

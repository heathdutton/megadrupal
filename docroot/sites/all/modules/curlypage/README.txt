
-- SUMMARY --

The Drupal curlypage module allows you to add pagepeel, "magic corners"  type
banners to your website. 

For a full description of the module, visit the project page:
  http://drupal.org/project/curlypage

To submit bug reports and feature suggestions, or to track changes:
  http://drupal.org/project/issues/curlypage


-- REQUIREMENTS --

None.


-- INSTALLATION --

* Install as usual, see http://drupal.org/node/70151 for further information.


-- CONFIGURATION --

* Configure user permissions in:

    Administer >> Configuration and modules >> People and permissions
    >> permissions >> Curlypage

      - administer curlypages

        Users in roles with the "administer curlypages" permission will be able
        to administer curlypages.


* Configure, add or delete your curlypages in:

    Administer >> Structure >> Curlypages


-- TROUBLESHOOTING --

* If you find any problem, write an issue in the project issue queu:

  - http://drupal.org/project/issues/curlypage

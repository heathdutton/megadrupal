NOTE: This module require account from online service to be used.

DESCRIPTION:

This module allows you to configure rules for use in prossess of
Drupal Commerce orders.

********************************************************************
INSTALLATION:

0. Create an account on one of those sites:

  - www.e-conomic.net
  
1. Download the module and extract it to used install directory.

2. Enable the module

3. Fill the service account information on the page here:

  Admin > Commerce > Config > e-conomic

4. Create a new filed on the shop order details, for use to compair
   debitors, called "phone number"

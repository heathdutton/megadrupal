OVERVIEW
========
READMORE.JS is a smooth, lightweight jQuery plugin for collapsing and expanding long blocks of text with “Read more” and “Close” links.

Note that this module requires jquery 1.7.x

INSTALL
=======
- Download plugin from https://github.com/jedfoster/Readmore.js
  and place under libraries/readmore_formatter, so you will have file in
  libraries/readmore_formatter/readmore.min.js
- Download Libraries API and enable module
- Download jQuery Update and enable module

What it does
===========
Wizard creates specialized add forms for the node add pane.

How to use
=========

1. Visit admin/structure/wizards
2. Click add
3. Pick a title
4. Pick a node type to use
5. Define steps. Steps are in the format of Title: [title] | fields:[[field_name],[field_name2]]
    title: Title | fields: [title]
    title: Space Type | fields: [field_oa_space_type,oa_parent_space]
    title: Space Access | fields: [group_access]
6. visit a panel page
7. Add a new pane, select Open Atrium then Add content.
8. Select your wizards and save.

Users will now see the wizard with each step separated when adding content from that panel.

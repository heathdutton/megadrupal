Overview
========
This module provides Addemar integration with the E-mail Marketing Framework
module.

Install
=======
- Enabled the E-mail Marketing Framework and the EMF Addemar modules
- Visit admin/settings/addemar to enter your Addemar configuration
- Wait for, or run, your site cron
- Configure list settings (see EMF's documentation)

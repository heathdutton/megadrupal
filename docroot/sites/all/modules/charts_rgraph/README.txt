Issues: http://drupal.org/project/issues/charts_rgraph

Sponsored by Relinc: http://www.relinc.it

Module to integrate RGraph HTML5 charts library.
RGraph is a canvas and JavaScript based charts library built for web charts.
RGraph is truly Open and Free to use for any purpose under the MIT license.

It requires the Charts module (http://drupal.org/project/charts) and supports
Views through it.

Currently supported chart types are pie, bar, column, line and area.


Installation
------------

 * Copy the whole charts_rgraph directory to your modules directory
   (e.g. DRUPAL_ROOT/sites/all/modules) and activate it in the modules page

 * Download RGraph from http://www.rgraph.net/download and put it in the 
   libraries folder, taki care that rgraph folder shall be in small letters
   (e.g. DRUPAL_ROOT/sites/all/libraries/rgraph).
 
 * Go to Charts module configuration page (at /admin/config/content/charts)
   to set RGraph as default graphic library for new charts.

 * Switch any chart already created to use RGraph as graphic library.

 * enjoy ;)

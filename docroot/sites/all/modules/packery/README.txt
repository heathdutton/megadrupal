This module integrates Packery with Drupal Views.

Requirements
 - Views
 - Libraries

Installation
Download the packery library from http://packery.metafizzy.co/

Place packery.pkgd.min.js in the library directory at; sites/all/libraries/packery
sites/all/libraries/packery/packery.pkgd.min.js

ImagesLoaded support
For imagesLoaded support please download the plugin from, https://github.com/desandro/imagesloaded
Place the plugin in the libraries directory at; sites/all/libraries/imagesloaded
sites/all/libraries/imagesloaded/imagesloaded.pkgd.min.js

More info about imagesLoaded, http://packery.metafizzy.co/appendix.html

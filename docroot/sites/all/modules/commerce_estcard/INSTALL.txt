------------------------------
         INSTALLATION
------------------------------

Download the module code.

Unpack into your modules directory.

Enable module.

Go to Payment methods admin/commerce/config/payment-methods

Enable Estcard (Nets Estonia) payment method

From same page edit the settings Operations->Edit

Appears page: Editing reaction rule "Estcard (Nets Estonia)"

Under Action choose Edit

Fill in Settings:
* Estcard URL
* Merchant ID
* Public Key
* Private Key
* Certificate

Save

Configure your Checkout settings admin/commerce/config/checkout

Enjoy


-- SUMMARY --

The bit.ly node module provides bitlinks for nodes.

For a full description of the module, visit the project page:
  http://drupal.org/node/2314739

To submit bug reports and feature suggestions, or to track changes:
  http://drupal.org/project/issues/2314739


-- REQUIREMENTS --

bitly.


-- INSTALLATION --

* Install as usual, see http://drupal.org/node/70151 for further information.


-- USAGE --

* Just enable the module and it'll create bitlink on node create/update or view.

* The short bitlink will be available in node.tpl.php file as $node->bitlink.

* Once the node is deleted bitlink is also deleted.

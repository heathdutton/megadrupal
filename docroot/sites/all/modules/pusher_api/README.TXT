This is the Pusher API.
To use this module you have to download the Pusher-PHP library from https://github.com/squeeks/Pusher-PHP. You can also download this librarie via "drush pusher_download"
To work the Pusher.php file has to be placed in the sites/all/libraries/pusher_api folder.

An example of a implementation of this module is included in the "Pusher API shoutbox" sub-module. You can use this module as a reference.

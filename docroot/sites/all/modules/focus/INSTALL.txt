REQUIREMENTS
------------

  * Drupal 7.0 or later

INSTALLATION
------------

1. INSTALL FOCUS

   Unpack the module in your sites module directory. Then browse to
   "Administration" -> "Modules" and enable the module.

3. CONFIGURE FOCUS (optional)

   Browse to "Administration" -> "Configuration" -> "User interface" ->
   "Form field focus" to add or remove forms that get focus.

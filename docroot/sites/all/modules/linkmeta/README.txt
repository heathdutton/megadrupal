INTRODUCTION
------------

This module creates a text format filter. The filter adds metadata to links.

Features:
 * Append file size to link.
 * Append extension to link.
 * Append extension based icon to link. Icon uploads appear to configuration
   page as new icons are displayed in content.

INSTALLATION
------------

1. This module requires file module from Drupal core.

2. Copy linkmeta/ directory to your sites/SITENAME/modules directory.

3. Add the features of this module to desired text formats 
   admin/config/content/formats.

4. Enable the module and configure admin/structure/linkmeta.

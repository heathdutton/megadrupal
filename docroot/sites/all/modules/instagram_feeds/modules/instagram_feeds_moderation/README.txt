DESCRIPTION
-----------

This module allows administrators to manage Instagram Feeds and Instagram Media
Items in user-friendly interface. It creates 2 views (for feeds and for images)
that are available in Admin > Content > Instagram (admin/content/instagram).

REQUIREMENTS
------------
    * Instagram Feeds

INSTALLATION
------------

1. Just enable the module after the Instagram Feeds installation.

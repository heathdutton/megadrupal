For adding swf files to your site add a file field to your content type
and in manage display tab select "Flash" formatter for this field

For adding games from miniclip.com go to setting page of module and
select content types that you want for showing games in them
and add a text field to this content types
then in manage display tab select "Miniclip game" formatter
for this field now you can paste url of the game page in this field
and the game will be displayed in your node

Registration codes module allows you to create special codes that allow users
to register in your site with. You can make the registration code required,
which makes your site practically invite-only. 

Included sub-modules allow you to add additional integration with role-based
registration codes, vouchers, and Organic Groups integration. 
 
Install the module: https://www.drupal.org/documentation/install/modules-themes

After installing, go to admin/config/people/regcode page for the options.

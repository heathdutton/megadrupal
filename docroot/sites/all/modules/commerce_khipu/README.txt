Commerce Khipu
==============

This module implements Khipu payment services for use with Drupal Commerce.

Dependencies:

1) Khipu
2) Drupal Commerce

Installation
-----------

Enable the module and then go to the payment methods settings
(admin/commerce/config/payment-methods). Enable the method "khipu" and
configure it from the edit link. Then edit the action to configure:

1) The khipu account to receive payments.
2) The subject that will be displayed on the page of khipu.
3) Optionally can define a description, could be about the store.
4) Optionally can define a url of an image to be displayed along with the
   description.
5) Button image to display on the form redirect.

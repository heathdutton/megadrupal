DESCRIPTION
===========

jRumble is a jQuery plugin Drupal module that rumbles, vibrates,
shakes, and rotates any element you choose.

CONFIGURATION
=============

* Go to the configuration screen at 
Administration >> Configuration >> Media >> jRumble.

Selector
--------

* The default selector is "#site-name". Enter your own 
jQuery selectors. Use a new line for each selector.

Style
-----

The Style of the Animation effects.
The list of animation style available are
   1. Light (Vibrate)
   2. Heavy (Vibrate)
   3. Shake
   4. Rotate

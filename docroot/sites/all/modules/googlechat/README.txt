-- SUMMARY --

This chat module enables you to seamlessly integrate Gmail style chat
into your existing website.

For a full description of the module, visit the project page:
  http://drupal.org/project/googlechat

To submit bug reports and feature suggestions, or to track changes:
  http://drupal.org/project/issues/googlechat

REQUIREMENTS
============
- This module is written for Drupal 7.x.
- The 7.x branch of jQuery Update module (jquery_update).


INSTALLATION
============

1. Download the module from Drupal.org/project/googlechat and save it to your
   modules folder.
2. Enable the GoogleChat module at admin/build/modules.
3. Set the access Googlechat permissions for authenticated users at
   admin/user/permissions.

COMMING UP FEATURES
===================

1. Optional integration with Organic Group
2. Custom status message
3. Block user
4. Node.js integration

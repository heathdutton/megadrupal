-- SUMMARY --

Views Record Count, a Drupal module, as the name says, count the records in
views. For displaying the count you have to enable the pager in you views
than you can also choose the region to display the count text "header/footer".

-- REQUIREMENTS --

Views Module (in Drupal Contrib) must be enabled.

-- INSTALLATION --

See the installation guide here - http://drupal.org/node/70151

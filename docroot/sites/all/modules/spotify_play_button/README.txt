  ------------------------------------------------------------------------------------
                         		ABOUT
  ------------------------------------------------------------------------------------

This module defines a new field type "Spotify Play Button", for embedding spotify tracks into your web page. 

See https://developer.spotify.com/technologies/spotify-play-button/ for more info on the play button.


  ------------------------------------------------------------------------------------
                         		USAGE
  ------------------------------------------------------------------------------------

Enable the module as usual.

When enabled, you will have a new field type "Spotify Play Button". 

It has two widgets: single (for adding one URI at a time), or Multi Track (so you can create a playable list of 
tracks without using a Spotify playlist by specifying a title and a comma-separated list of track IDs).

You can customise the field display to show either a large or compact player. If large, you can also change the 
theme colour (black or white) or view (list or coverart)

  
  
  ------------------------------------------------------------------------------------
                         	   REQUIREMENTS
  ------------------------------------------------------------------------------------
  
The only requirements is the field module.

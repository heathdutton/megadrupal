Ubercart Cart Edit Links

Adds the possibility to return from cart to the
product form and change the selected options.

Once you enable the module, the user will be presented
an edit link on the cart page.Clicking on it brings
the user back to the product page, with his previously
selected options. He may now change these options and
the quantity and save his changes to the cart product.

Installation:
Just enable the module, it will work out of the box.

INTRODUCTION
------------
This module adds custom icon packages to Drupal File field.

 * For a full description of the module, visit the project page:
   https://drupal.org/project/file_field_icons
 * To submit bug reports and feature suggestions, or to track changes:
   https://drupal.org/project/issues/file_field_icons


REQUIREMENTS
------------
This module requires only the File(core)


INSTALLATION
------------

* Install as you would normally install a contributed Drupal module. See:
   https://drupal.org/documentation/install/modules-themes/modules-7
   for further information.


CONFIGURATION
-------------
 * Configure user permissions in Administration » People » Permissions:
   - Administer file icons
     Allow users to change file icons packages.
     
 * Customize the icon package in Administration » Configuration »
   Media » File Field Icons


USAGE
-------------
You can use default icon packages included in this module or, you can create
your custom icon packages by:
- create a 'file_field_icons' folder inside your files directory.
  (eg: 'sites/default/files/file_field_icons')
- clone the 'Sample' folder located in the module root directory in
  the folder you just created.
- go to the File Field Icons administration page and select your package.

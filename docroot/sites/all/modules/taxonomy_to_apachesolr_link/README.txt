Instructions:
This module changes any taxonomy term link to be used as a link to
an Apache Solr search filter.
To Change the taxonomy link:

1. Click the "manage display" tab for the content type you desire
to change the term render.
   Example: article node type: admin/structure/types/manage/article/display
2. Locate the taxonomy term and Select the new Format called Apachesolr Link
3. Optional: Adjust the path and target with the cogwheel to the right.

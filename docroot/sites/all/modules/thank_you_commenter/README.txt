-- SUMMARY --

Thank you commenter, a drupal module, as the name says, thanks the commenter by
sending them a thank you mail. You can choose to send mail to registered user or
anonymous user. You must select the content types on which these settings will
be implemented.

-- REQUIREMENTS --

Comments Module (in drupal core) must be enabled.

-- INSTALLATION --

See the installation guide here - http://drupal.org/node/70151

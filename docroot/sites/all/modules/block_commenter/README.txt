-- SUMMARY --

This Block Commenter Module will provides a flexible method for Drupal site
administrator to block the list of authenticated users from commenting
corresponding to selected content types.
Thus it prevents spammers from commenting on the Drupal site.

We also plan to add
1. Different setting form for Editors.
2. Per Role base blocking.


-- REQUIREMENTS --

Comments Module (in drupal core) must be enabled.


-- INSTALLATION --

See the installation guide here - http://drupal.org/node/70151

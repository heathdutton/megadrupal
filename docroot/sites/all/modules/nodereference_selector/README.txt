Node Reference Selector Widget was created by Mihhail Arhipov

Dependencies
------------
 * Node Reference (part of References)
 * Image (part of core)

Install
-------
1) Put the module directory (nodereference_selector) into sites/all/modules 
folder in your Drupal installation.

2) Enable the module in admin/modules or using drush

3) Create an Image Style, which will be used to display the referencing node

4) Add a Node Reference field to a content type

5) Select content types you wish to reference (These content types must have an 
Image field attached to them)

6) Configure the widget by selecting an Image Style and an Image field for 
each content type.

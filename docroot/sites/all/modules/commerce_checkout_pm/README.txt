-- SUMMARY --

Provides a UI to manage checkout pages.
After enabling the module, go to admin/commerce/config/checkout/pm
and configure checkout pages.

-- FEATURES --

Allows you to change the page title, help, submit value and disable back button.

-- REQUIREMENTS --

https://drupal.org/project/commerce

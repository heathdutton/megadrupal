-- SUMMARY --

Integrates the DropKick plugin with Drupal.


-- INSTALLATION --
	
	1. Download and unpack the Libraries module directory in your modules folder
     (this will usually be "sites/all/modules/").
     Link: http://drupal.org/project/libraries
  2. Download the DropKick (http://robdel12.github.com/DropKick/) jQuery plugin
     and extract the file under sites/all/libraries. Make sure the path to the
     plugin file becomes: "sites/all/libraries/DropKick/dropkick.js"
  3. Download and enable the module.


-- CONFIGURATION --
	1. Configure at Administer > Configuration > User interface > DropKick.
	2. For mobile device support enable mobile support checkbox.
	3. For IE8 support enable the IE8 support checkbox.
	4. You can change DropKick default theme to classic theme. These is
		 experimental feature.


-- DEPENDENCIES --

  1. Libaries API (http://drupal.org/project/libraries)


-- SUMMARY --

Provides the "Yandex.Share" extra field for nodes and terms,
with which you can add Share button to your pages.
Provides "Yandex.Share" block to display
the Share button in any region of the site.
Uses http://api.yandex.ru/share/

-- INSTALLATION --

Install as usual.
See http://drupal.org/documentation/install/modules-themes/modules-7
for further information.

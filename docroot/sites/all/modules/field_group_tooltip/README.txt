Requirements
------------------
1. Field group module (http://drupal.org/project/field_group).
2. BeautyTips module(http://drupal.org/project/beautytips).


Installation and configuration
------------------
1. Unpack and move directory "field_group_tooltip" to your modules directory.
2. Enable it in the modules list of your site.
3. Create field group of "Tooltip" type.
4. Move some fields into this field group.
5. Tooltip will appear on hovering over choosen entity.

Known issues
------------------
In clean Drupal install field group type of tooltip not showing on list.

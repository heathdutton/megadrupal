Translate set
-------------

This module allows you to collect a set of strings and translate those on one
page in the admin interface. This module was built to allow translators on a
website and present them one interface only having to translate the necessary strings
that are used on the front-end, mails, rules etc.

Features:

- Add or remove strings to the translate set via the standard translate
  interface. When searching strings, a link will be available in the operations
  column to add or remove this string to the translation set.
- Manage the strings in the translate set on a bulk overview page.
- Tracking of new strings to add to them automatically to the set.
- Per user configuration to restrict the languages the user can translate to at
  'user/{id}/translate-set'.

Permissions:

- translate set: allows access to 'Translate set' screen.
- manage translate set: allows bulk management of strings in the set.
- administer translate set settings: configure the settings.
- administer languages for translate set per user: allows restricting the
  languages a user can translate to.

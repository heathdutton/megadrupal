Better_Revisions
==========

Better revisions modifies the built-in Drupal revision tab to
include a configurabe select list of revision types as well as
an optional text area for detailed revision explanations.


Dependencies and incompatibilities
============

* None at present


Installation and Usage
======================

1) Copy the better_revisions folder to the modules folder in your installation.

2) Enable the module (/admin/modules).

3) Grant the 'administer better revisions' permission to selected roles and visit
   the configuration page at admin > config > content > better_revisions
   to choose the revision type options, inclusion of text are, titles and
   help text.
   
The standard revision text area will be replaced with your customized revision widget.

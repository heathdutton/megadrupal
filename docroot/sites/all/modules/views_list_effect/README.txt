Summary
-------
This module allows you to easily create CSS3 animated list in a view. Optionally, you can use waypoints module in order to trigger the animations when the view is visible.

Module Features
---------------
You can choose between 23 effects.
Apply random behaviour for the effects.
Waypoints integration.
CSS optimization: It loads only the css needed.

Configuration
-------------
Select the new list effect style in your view.
Configure type of animation effect, delay, duration and rest of behaviors.

Recommended
-----------
Integrates with waypoints module. You only need to download, enable it and mark "Use waypoints" option in the style settings.

Make sure you have https://getcomposer.org/ and
https://github.com/drush-ops/drush installed

Ensure you have downloaded all the module dependencies listed on the project page.

# drush en cointools cointools_fiat cointools_daemon --yes

There may be error messages here.

# drush composer-rebuild
# cd sites/default/files/composer
# composer install
# drush cc all

It may be necessary to change directory permissions.

## SUMMARY ##

This module extends the functionality of the insert module
by providing a convenient popup overlay on node edit pages.

Please make sure that you enable (only) one field with the
insert module enabled on it.

Requires the wysiwyg module and insert module.

## Installation ##

Simple enable and enjoy. There is no configuration for this
module (as yet).

Also, ensure that you enable the 'Inline image insert popup'
plugin that comes with this module in:
/admin/config/content/wysiwyg/profile/[FORMAT]/edit

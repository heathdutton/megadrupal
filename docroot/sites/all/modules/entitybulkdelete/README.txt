INTRODUCTION
------------

Entity Bulk Delete is a module which delete the entity in 
bulk using batch processing API. Entity Bulk Delete module
provide the option to delete the entity using customize way.
This module will permanently delete your entity so it is
advised to use that module with your own risk and make
sure you have a backup.


INSTALLATION
------------

The Entity Bulk Delete module is very similar to other Drupal modules.
For installation of the Entity Bulk Delete module 
please follow the below mentioned steps:

• Install as usual
• Enable the Entity Bulk Delete module.


CONFIGURATION
-------------

• There is no configuration for this module.

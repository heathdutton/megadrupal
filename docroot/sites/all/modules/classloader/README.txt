
-- SUMMARY --

Enables Symfony's PSR-0-compatible UniversalClassLoader for loading namespaced
PHP 5.3 classes.

For a full description of the module, visit the project page:
  http://drupal.org/project/classloader
To submit bug reports and feature suggestions, or to track changes:
  http://drupal.org/project/issues/classloader


-- REQUIREMENTS --

* PHP 5.3.3


-- INSTALLATION --

* You normally should not need to install this module, unless another module
  depends on it.

INTRODUCTION
------------
prettyPhoto is a jQuery based lightbox clone. Not only does it support images,
it also add support for videos, flash, YouTube, iFrame. It's a full blown
media modal box.

 * Provides prettyPhoto formatter for Image module.
 * Provides prettyPhoto formatter for YouTube Field module.
 * Provides prettyPhoto formatter for Entity reference module.
 * Provides prettyPhoto formatter for Link module.

REQUIREMENTS
------------
This module requires the following modules:
 * Libraries API (https://drupal.org/project/libraries)
 * The jQuery prettyPhoto plugin

INSTALLATION
------------
 * Download prettyPhoto v3.x from https://github.com/scaron/prettyphoto
 * Uncompress the downloaded file (jQuery prettyPhoto plugin), and copy the 
 "js" and "css" folders into "/sites/all/libraries/jquery.prettyPhoto" folder.
 * Install prettyphoto_formatters module as described here
 https://drupal.org/documentation/install/modules-themes/modules-7
 * View the status report at admin/reports/status to check that prettyPhoto
 is installed correctly.
 * Visit admin/config/media/prettyphoto-formatters to adjust the settings
 * Select a prettyphoto formatter in field display settings

README - PINIT
**************************

SHORT PROJECT DESCRIPTION
--------------------------
This module allows to integrate Pinteres't Pin It Button in our sites.

Basically, the module fetch the image, url and description from the node 
and create a link for share on Pinterest.

Pinterest?
Pinterest is a social bookmarking tool used to “pin” images found around
the Web into categorized collections, or boards.


INSTALLATION
------------
1. Place the pinit directory into your Drupal sites/all/modules/ directory.
2. You can configure it on: admin/config/content/pinit

Overview
========

The jQuery selectBox module is a styleable replacement for SELECT elements using
Cory LaViska's jQuery UI plugin

You can view a demo of the plugin here:

http://labs.abeautifulsite.net/jquery-selectBox/

Requirements
============

 * Drupal Libraries module (http://drupal.org/project/libraries)
 * Drupal selectBox module (https://drupal.org/project/selectbox)
 * jQuery selectBox (https://github.com/claviska/jquery-selectBox/downloads)

Installation
============

1) Upload/install the Libraries API module
2) Create a sites/all/libraries directory on your server
3) Create a directory within sites/all/libraries named jquery.selectbox
4) Download the latest jQuery selectBox plugin from
   https://github.com/claviska/jquery-selectBox/downloads
5) Upload/install the jQuery selectBox plugin: place it inside the
   jquery.selectbox directory
6) Enable the jQuery selectBox module

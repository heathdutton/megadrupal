Description
-----------
This module controls the provisioning of Shibboleth authentication access
when new accounts are created, allowing admins to set accounts up in advance
before a login via Shibboleth has occurred.

Requirements
------------
Drupal 7.x
shib_auth 7.x-4.0

Installation
------------
1. Copy the entire shib_provision directory into Drupal's sites/all/modules 
   (or sites/default/modules) directory.

2. Login as an administrator. Enable the module in "Administer" -> "Modules"

------------------------------------------------------------------------------------

openbadging assertion baker is a validation module for earned badge. Anyone can 
validate a badge through this module. Every badge has a url in badge image meta 
data, and that URL contain all information about that badge. User can see any 
image meta data using any tool for 
example(http://www.fileformat.info/convert/image/metadata.htm 
a online meta viewer). 
In properties section of meta data a url contains title openbadges appear. 

Installation
------------

* Copy the whole "openbadging_assertion_baker" directory to your 
  modules directory (sites/all/modules).
* Enable the module from (admin/modules).


-----------------------------------------------------------------------------------------------------------------

Adds a "forward this page" link to each bundle of openbadging entity. 
You can customize the default form field values.


Installation
-------------

* Copy the whole "openbadging_forward" directory to your
  modules directory - "sites/all/modules"
* Enable the module "admin/modules".
* Set your desired permissions at - "admin/people/permissions".


Usage
------
   
 * Go to "/admin/config/user-interface/openbadging_forward" to 
   configure the module. This path is also linked from the Configuration page 
   and the Modules list page within site administration.
 * For show forward link on any content tick bundle in openbadging forward 
   configuration and save it. 

General Description
--------------------

* Goto "/admin/config/user-interface/openbadging_forward" and  tick badges 
  bundle and selsct icon and text and save it.

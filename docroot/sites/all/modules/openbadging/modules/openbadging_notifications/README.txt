This module sends an e-mail on each give openbadging action.



Installation
-------------

 * Copy the whole "openbadging_notification" directory to your 
   modules directory - "sites/all/modules" and
   enable the module from path "admin/modules".
 * Go to "/admin/structure/openbadging/notification" and save Configurations.


Usage
-----
   
 * Go to "/admin/structure/openbadging/notification". Here you will see the 
   Notification template - 
   "structure >> Openbadging Configurations >> Notification Templates."
 * You can edit mail format with notification templates.

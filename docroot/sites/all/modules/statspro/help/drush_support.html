The Statistics Pro's Drush support implements the following command:

statspro
Usage: drush [options] statspro [alert|aliases|comments|critical|emergency|error|modules|nodes|node_types|pi|sessions|terms|ualert|ucritical|uemergency|uerror|upi|users|uwarning|warning]

statspro

The statspro command returns the value of the select mode for today.

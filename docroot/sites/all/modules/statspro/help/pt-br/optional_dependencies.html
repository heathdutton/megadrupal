O módulo Statistics Pro pode aproveitar diversos módulos opcionais através dos quais disponibiliza recursos adicionais.

Segue aqui a lista dos módulos opcionais reconhecidos pelo Statistics Pro assim como relações de recursos adicionais relacionados a cada um deles.

Statistics (módulo central do Drupal)

Com o módulo central Statistics o módulo Statistics Pro disponibiliza dois relatórios:

  • relatório de acesso agregado por URLs
  • páginas apresentadas

Charts and Graphs

Com o módulo Charts and Graphs o módulo Statistics Pro produz gráficos para os seguintes relatórios:

  • visão geral
  • relatório de acesso agregado por URLs

Views Charts

Incluíndo o módulo Views Charts junto com o Charts and Graphs permite que o Statistics Pro to produza gráficos para os seguintes relatórios:

  • comentários
  • logs
  • nós
  • páginas apresentadas
  • usuários

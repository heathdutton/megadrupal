Statistics Pro can take advantage of several optional modules with which it provides extra features.

Here is a list of these modules followed by a description of the extra features associated with each one.

Statistics (core)

With the core Statistics module Statistics Pro enables two extra reports:

  • path aggregated access report
  • page visualization report

Charts and Graphs

With the Charts and Graphs 6.x-2.0+ module Statistics Pro enables graphs for the following reports:

  • overview
  • path aggregated access report

Views Charts

Including the Views Charts module together with Charts and Graphs enables Statistics Pro to provide graphs for the following reports:

  • comments
  • logs
  • nodes
  • page visualizations
  • users

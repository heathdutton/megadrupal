  ------------------------------------------------------------------------------------
                           ABOUT TOOLS
  ------------------------------------------------------------------------------------

	This provides a basic interface for turning on and off features & modules, suitable for non-techy users.
	
	It uses the switch module to provide an iphone-like button for turning on/off items. 

  ------------------------------------------------------------------------------------
                           USAGE
  ------------------------------------------------------------------------------------

	You can set a module to appear in the tools page, by adding tool = true to the module's .info file.

  ------------------------------------------------------------------------------------
                         INSTALLATION
  ------------------------------------------------------------------------------------
  
  	No special installation requirements, just download it to your module directory and turn it on.
  
  ------------------------------------------------------------------------------------
                         REQUIREMENTS
  ------------------------------------------------------------------------------------
  
	The following modules are required:
	
	  * features
	  * switch

Panels Variant Page Title allows you to set your page title directly from Panels
configuration, and also allows you to take advantage from Panels Arguments and
Contexts as Placeholder tokens.

Panels Variant Page Title configuration saved with the Panels configuration
so it is exportable with Features module.

Installation
1. Unpack the module in your Drupal sites/all/modules directory.
2. Enable the module in Admin menu > Site building > Modules (admin/modules).

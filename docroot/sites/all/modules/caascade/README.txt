I. Introduction

Caascade allows users to request output from the Maxima CAS.

II. Requirements

1. Drupal 7
2. Caascade & MathJax modules

III. Setup

1. Download and enable this module on your web server.
2. Enable the desired operation block.

IV. Customization

- To override the CSS or JS file place a copy in html/override.
- Visit the Caascade module settings page under Web Services to customize.
- To modify block content, copy .HTML files to the override directory.
- Hide some or all blocks to reduce clutter

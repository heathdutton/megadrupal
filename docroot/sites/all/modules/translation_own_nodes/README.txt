TRANSLATION OWN NODES
=========================
Translation own nodes extends Drupal support for translation nodes by author. This is inspired to implement simple solution in add a new user permission "Translate own nodes" (changes in the core drupal)(http://drupal.org/node/253157#comment-1153121) as a new module.

Install
=========================
1) Simply drop this module's directory into your modules directory 
2) Install dependences (Content translation)
3) Install via admin/build/modules.

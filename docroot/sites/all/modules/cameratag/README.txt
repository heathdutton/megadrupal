Addressfield autocomplete
------------------------------------------------------------------------------
This modules provides integration with the cameratag service. It provides a
field fomatter that can be used to record videos from the users computer
and display the videos using the cameratag service.

Installation
------------------------------------------------------------------------------
1. Install the cameratag module
2. Navigate to admin/config/cameratag and add your own Cameratag API Key and
   Camera uuid. They can be found in the cameratag settings dashboard.
3. Navigate to your content type / profile / form and create the field
   choosing the widget Cameratag Field.
4. Provide the settings for the field.
5. Save the field. Cameratag should now display in the form.

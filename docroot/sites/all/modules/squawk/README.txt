WHAT?
-----
Enabling this module will cause your web site visitors to make a single
request to a random IP address, for every request that you serve with the
script tag, in order to add noise to the logs being kept by Australian ISPs.

HOW?
----
Extract the module to sites/all/modules and enable Squawk via the modules
administration page.

WHY?
----
In order to add noise to the logs being kept by Australian ISPs.

MORE?
-----
See http://squawk.cc/

AFFILIATE STORE LIGHTBOX
------------------------

This module provides lightbox integration for affiliate store. When this module
is enabled, product full image will be displayed in a nice lightbox popup.


REQUIREMENTS
------------

In order for the lightbox to appear, you must install at least one of the
following lightbox type modules:

- Lightbox2:
  http://drupal.org/project/lightbox2

- Colorbox:
  http://drupal.org/project/colorbox


CONFIGURATION
-------------

Configure the module settings at:
  Administration > Structure > Affiliate store
There will be extra options under product advanced settings.

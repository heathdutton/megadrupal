CiviCRM Reference Fields


Description:
-------------------------------------------------------------------------------
This is CiviCRM-Drupal integration module.
Module allow to create following fields of type 'Select', 'Radio', 'Check-box', 
Autocomplete 
# CiviCRM Contribution page reference field with different start date, end date 
  criteria.
# CiviCRM Event Page reference field with different event start date, end date 
  criteria.
# CiviCRM PCP page reference field.


Requirements:
-------------------------------------------------------------------------------
Requires Drupal core modules: Field, List, Text.
And CiviCRM installation (http://civicrm.org/).


Installation & Use:
-------------------------------------------------------------------------------
1. Enable module in module list located at administer > structure > modules.
2. Go to field settings of the content type
   (eg: admin/structure/types/manage/page/fields).
3. Add field(s) of type CiviCRM Contribution pages/CiviCRM Event Pages/CiviCRM 
   PCP pages.
4. Field(s) will be avialble on create/edit content of that content type
   (eg: node/add/page).

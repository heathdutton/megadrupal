MAIL SIGNATURE
==============

This module simply gives admin a way to add
a signature to every mail sent by the system.


INSTALLATION
============
- Download the module
- Install it with drush or from admin/modules

USAGE
=====
- Go to admin/config/system/mail-signature
- Add a signature
- Check the flag to activate the signature
- Test signature
- Use drush to enable or disable signature

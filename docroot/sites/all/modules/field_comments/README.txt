This module allows commenting on fields for any field display.

Comments on particular field can be enabled
at the Manage display settings for content types.

Instructions:
The field comments settings are found
in the Manage display tab for content types.
A text checkbox is available for each field, revealed by using
the formatter settings edit button (Gear wheel icon) for that field.

Dependencies:
Field formatter settings API

Module: Virtual Sites
Webpage: http://drupal.org/project/virtual_site
Author: Fokke Zandbergen (fokke@zo.nu)

************************ About

Virtual sites offers almost the same (and more) functionality as the Drupal multi-site feature without the need for FTP 
or SSH. Depending on conditions (e.g. requested url or user role) handeld by the Condition(s) module, you can override 
theme, site information, menu's and more to virtually present the visitor with a different website.

************************ Features & Settings

Check: http://drupal.org/project/virtual_site

************************ Installation

1. Install and enable the conditions module (http://www.drupal.org/project/condition).
2. Copy this directory to your site's modules folder.
3. Enable the module at admin/build/modules.
4. Configure access control to the admin pages at admin/user/permissions.
4. Create one or more conditions you can use to determine when a virtual site is used.
5. Create and configure virtual sites at admin/build/site.

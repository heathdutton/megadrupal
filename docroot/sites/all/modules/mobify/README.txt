Mobify is pleased to provide a Drupal module!

Use of this plugin requires a configured custom domain (CNAME).

   1. Download the Mobify Drupal Module.
   2. Follow the Drupal module installation guide (http://drupal.org/node/70151)
   3. Once activated, configure the module under "Site Configuration" and then "Mobify". Enter your configured custom domain (CNAME).

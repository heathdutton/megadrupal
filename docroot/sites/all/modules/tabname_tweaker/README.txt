
Description
------------

The Tab Name Tweaker module allows admins to override the tab name displayed when viewing a user profile page or node page. The default tab name is "View".
This module can be especially useful for Organic Groups, where the normal 'View' tab serves as an overview or landing page rather than a specific piece of content.


Installation
------------

1. Place the module in the modules directory for your site and enable it.
2. Configure the user account tab name and tab names for all content types at /admin/structure/tabname-tweaker
   or configure tab names for specific content types by navigating to the edit tab of your content type at /admin/structure/types

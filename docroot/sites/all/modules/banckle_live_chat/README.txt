Overview
========

Banckle Live Chat module seamlessly integrates Banckle Live Chat Deployment
Widget into your Drupal site.

Features
========

  * Admin can input account info and select a deployment
  * Admin can customize some features of the Deployment widget
  * Admin can allow users to disable chat button
  * Admin can select which pages to skip when displaying button
  * Users can hide chat button

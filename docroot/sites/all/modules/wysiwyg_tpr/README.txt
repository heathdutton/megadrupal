
-- SUMMARY --

WYSIWYG toolbar per role is an extension to Wysiwyg API that allows you to
configure editor settings per role.

For a full description of the module, visit the project page:
  http://drupal.org/project/wysiwyg_tpr
To submit bug reports and feature suggestions, or to track changes:
  http://drupal.org/project/issues/wysiwyg_tpr


-- REQUIREMENTS --

* Wysiwyg (https://drupal.org/project/wysiwyg).


-- INSTALLATION --

* Install as usual, see
  http://drupal.org/documentation/install/modules-themes/modules-7



-- CONFIGURATION --

* Go to Administration » Configuration » Content authoring » Wysiwyg,
  and edit the role-specific settings for your editor profiles.

INTRODUCTION
------------

 * Easy currencyCON is a currency convertor made available for all the Drupalers who wish to use currency convertor module in their website.

INSTALLATION
------------
 * Install it as you would normally install a contributed drupal module. See https://drupal.org/documentation/install/modules-themes/modules-7 

HOW TO USE
----------
 1) Go to admin/modules and install the module EasyCurrencyCON.

 2) Once this module is installed, block is created, and you can place this block in the way you want to see on UI.

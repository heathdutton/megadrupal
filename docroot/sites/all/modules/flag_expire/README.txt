Flag expire
============

This module allows flags to be set to expire after a set duration or on at a
specific time.

Requirements
------------

In addition to Flag, one of the following is required:

- Date API (part of the Date package, http://drupal.org/project/date)
  Additionally, Date module is required to have expiry dates per flagging.
- Time Period (http://drupal.org/project/timeperiod)

Configuration
-------------

New expiry options are added to the flag admin form.

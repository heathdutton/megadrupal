Views glossary module provide following features,

1- A special kind of layout to glossary view(which is available by
default with views) or any view, see in screenshot.
2- Providing pager with selectbox and textbox.
3-A default view is added for example.

Usage,
Go to module configuration(admin/config/views_glossary_settings)
and enter view machinenames separated by comma.
See the layout on view.

Note:
This module will work only with views ajax disabled and views display of page.

Webform Node Element


== ABOUT ==

This module allows you to include node content as markup in a webform without
using the PHP input filter. Enter a node ID to let the module display the node
body either in full or as teaser.


== INSTALLATION ==

Unpack the module into sites/all/modules and enable it. You cna now add node
components to your webform.


== EXTRAS ==

If the ctools module is available, you will be able to select nodes via the
the ctools auto-complete UI.

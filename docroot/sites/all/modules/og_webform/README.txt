Description
-----------
This module provides integration between Organic Groups and Webform  modules. It
uses the og_permissions system to add webform permission settings to OG's
permission list.

Requirements
------------
Drupal 7.x
Organic Groups
Webform

Installation
------------
1. Copy the entire og_webform directory the Drupal sites/all/modules directory.

2. Login as an administrator. Enable the module in the "Administer" -> "Site
   Building" -> "Modules"

3. Assign permissions to the Organic groups global permissions settings,
   (admin/config/group/permissions) or on each group's permissions page.

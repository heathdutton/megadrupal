INTRODUCTION
------------
This module contains the following extras for Node expire module.
 * "Update expiry date" Rules action. Configuration is located at 
   "Structure -> Content types -> {Your Content type} -> Publishing options".
 * Tokens support. Tokens [node:expiry] and [node:expired] are available.


REQUIREMENTS
------------
This module requires the following modules:
 * Rules (https://drupal.org/project/rules)
 * Node expire (https://drupal.org/project/node_expire)


INSTALLATION
------------
 * Install as you would normally install a contributed drupal module. See:
   https://drupal.org/documentation/install/modules-themes/modules-7
   for further information.

Copies gravatars to a local cache and applies the user-picture image
style to them before display.


INSTALLATION

 * Extract to sites/*/modules.

 * Enable via the modules page.

 * Choose a picture display style for user pictures on
   the Account settings page.

 * Profit.


REQUIREMENTS

 * Image (in core)
 * Gravatar - http://drupal.org/project/gravatar

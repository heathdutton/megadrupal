INTRODUCTION
------------

QueryBuilder is a module which build dynamic query for developers. 
QueryBuilder module helps the drupal developer especially those 
guys which are fresher in drupal.It reduces the amount of time for 
building query. It also provide options to clear cache, 
every time when cron run.


INSTALLATION
------------

The QueryBuilder module is very similar to other Drupal modules.
For installation of the Query Builder module 
please follow the below mentioned steps:

• Install as usual
• Enable the QueryBuilder module.


CONFIGURATION
-------------

• There is no configuration for this module.

Stamps.com Shipping Labels

This module allows you to use the Stamps.com® software to print USPS shipping
labels from Ubercart. It is an Ubercart shipping method that imports and
exports Stamps.com® XML files for batch processing.

As of this writing (March 2013), this module is currently the only open source
method for printing USPS shipping labels for Ubercart orders

See http://drupal.org/node/1952576 or the online help page for this module
for full documentation.

* README: *

DESCRIPTION:

This module create a new field type "Paypal field" that gives a payment
option to any content type.


INSTALLATION:

1. Extract paypal_field into your Drupal sites/all/modules/ directory.

2. Enable the email module by navigating to: admin/modules


USAGE:

Go to a Manage fields on a content type or an other entity, you will now
have a new field type called "Paypal field".

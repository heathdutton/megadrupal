INTRODUCTION
------------

The 7.x version of the Taxonomy Autotagger module implements the interfaces 
created by the Tag API module, allowing a vocabulary to be searched against for 
suggestions of terms related to the content of a node.

REQUIREMENTS
------------
- Tag API

CONFIGURATION
-------------

- The module adds a configuration element to each node settings page.

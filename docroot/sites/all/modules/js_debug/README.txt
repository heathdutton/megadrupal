Welcome to the JS Debug module.

Summary
-------
Dump PHP variables and other debug info to the JavaScript console.


Requirements
------------
A web browser that has Console logging functionality.


Installation & Usage
--------------------
1. Install and enable JS Debug like every other Drupal module.
2. Pass a PHP variable into the function:

   js_debug_log($the_var);

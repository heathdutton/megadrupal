INTRODUCTION
------------

HTTP proxy provides an interface for configuring drupal_http_request proxy
settings.

REQUIREMENTS
------------

HTTP Proxy requires Drupal 7.17 or above.

INSTALLATION
------------

HTTP proxy can be installed by following the standard installation process
outlined at http://drupal.org/documentation/install/modules-themes/modules-7.

PERMISSIONS
------------

HTTP proxy provides one permission:

 * Administer HTTP proxy

USAGE
-----

Administrators or users with the Administer HTTP proxy permission are able to
configure proxy settings by visiting the HTTP proxy administration page.

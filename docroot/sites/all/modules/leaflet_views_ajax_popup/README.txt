This module provides an alternative views plugin for Leaflet module (https://www.drupal.org/project/leaflet)
with popups being rendered entities loaded via AJAX.

Installation
------------

1. Make sure Leaflet views module is installed properly.

2. Install Leaflet views AJAX popups as normal Drupal module

Usage
---------

Select "Leaflet Map (AJAX Popups)" as view display style and configure it the same way as "Leaflet Map"
style is configured.

The module displays views result entity as a popup, rendered with a view mode set in display style. E.g. if
a view returns list of users, popup will be rendered user entity, etc.

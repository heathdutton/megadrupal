Description
-------
The eBooks Search module reates a custom search that allows users to search eBooks from both Overdrive and 3M vendors.

This module is maintained by the Virtual Services team at Richland Library (http://www.myrcpl.com) in South Carolina.
The concept and coding for the Drupal 6 module were written by Suzane Goncalves (http://drupal.org/user/2359442).
The Drupal 7 version is maintained by Mark W. Jarrell (http://drupal.org/user/249768).

Install
-------
Installing the eBooks Search module is simple:

1) Copy the ebooks folder to the modules folder in your installation.

2) Enable the module using the Modules administration page (/admin/modules).

3) Give some of your users the ability to "Search eBooks" by going to the Permissions page (/admin/people/permissions).

4) Configure settings for the module on the eBooks Search Settings page (/admin/config/search/ebooks).

Support
-------
If you experience a problem with the module, file a
request or issue on the eBooks Search issue queue (http://drupal.org/project/issues/ebooks).
DO NOT POST IN THE FORUMS. Posting in the issue queue is a direct line of
communication with the module author.

1. Copy the countdown module folder to the modules directory.
2. Activate the module in Administer -> Configuration and Modules -> Modules (admin/config/modules)
3. Configure the block in Administer -> Structure -> Blocks -> Countdown configure (admin/structure/block/manage/countdown/0/configure).
   The block configuration is where to set the event name,
   accuracy of the output, and the target date/time.
4. Enable the block in Administer -> Structure -> Blocks

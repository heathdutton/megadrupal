I. Introduction

Quatriceps is an math API for self-learning that generates solutions to common problems.

II. Requirements

1. Drupal 7
2. MathJax and Quatriceps modules

III. Setup

1. Download and enable this module on your web server.
2. Enable the desired operation blocks

IV. Customization

- Override the JS and CSS files by placing a copy into html/override
- Visit the module's settings page under Web Services to customize
- To modify block content, copy .HTML files to the override directory.
- Hide some or all blocks to reduce clutter
- Settings can be adjusted at math.tetragy.com/user

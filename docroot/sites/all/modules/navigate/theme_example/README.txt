
--------------------------------------------------
Alternate Navigate Theme Instructions
--------------------------------------------------

To use an alternate theme for Navigate, follow these instructions:

- Copy the navigate-custom.css file and navigate-images folder to your
  theme directory

- Add the following line to your theme's .info file:
  stylesheets[all][] = navigate-custom.css
  
- Clear the cache to load the css. You should see Navigate with a
  lemony theme.
  
- From there, you can just tweak images and CSS to your liking. You
  can get a loader graphic here: http://www.ajaxload.info/

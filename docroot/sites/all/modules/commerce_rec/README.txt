==================================
Commerce Recommender Documentation
==================================

Installation & Configuration
----------------------------

Please first follow README.txt file in Recommender API 
(http://drupal.org/project/recommender) and correctly set up Recommender API.

After you enable Recommender API and Commerce Recommender, please go to 
admin/config/recommender and issue the command to run the Commerce Recommender. 
Then you can enable any of the 2 default views, and customise the views if 
necessary to display the recommendations.

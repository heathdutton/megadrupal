Commerce Reorder allows users to reorder past ordered items by providing a
'Reorder' button support for Views and also a Reorder tab for administrators.

Installation
------------
Enable the module, you should be able to see a Reorder tab in admin orders
screen.
If you want to add a Reorder button, edit the Commerce user orders view and add
the field Commerce Order: Reorder Button to it.

Known Issues
------------
"Save" button is displayed in Views: See http://drupal.org/node/1329520 for a
recommended method to hide the button if desired.

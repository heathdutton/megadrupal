Description
-----------
Current module is the extension of the basic drupal_sync module.
It allows to synchronize field-collections.

Installation
-----------
1) Enable the module. Then go to Settings page- admin/config/drupal_sync/settings
 on all sites that should be synchronized. Select all field_collection types being
synchronized  in "SYNC ENTITIES" section.
2) "Syncronization settings"  layout will appear on field_collection edit
pages- admin/structure/field-collections/%field_collection_item/sync_settings.  Here you can finish setting up
synchronization by customizing outer websites.

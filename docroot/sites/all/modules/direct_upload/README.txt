
-- SUMMARY --

Direct Upload allows to use drag&drop and copy/paste image upload in CKEditor.
  Images can be uploaded directly from local file system or from clipboard.

In this first version, only Scald image atom is supported.


-- REQUIREMENTS --

* CKEditor 4.5 (http://ckeditor.com/blog/CKEditor-4.5-Beta-Released)
* Scald 1.x-dev or 1.4 with dndck4 plugin (https://www.drupal.org/project/scald)


-- INSTALLATION --

* In the CKEditor profile configuration page
  (admin/config/content/ckeditor/edit/*), Editor Appearance section, enable both
  "Direct Upload" and "Scald Drag and Drop integration - CKEditor 4 widgets"
  plugins.

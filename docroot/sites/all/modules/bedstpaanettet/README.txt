
TOP OF THE WEB (BEDST PÅ NETTET) MODULE README
----------------------------------------------

This module is intended for participants in The Top of the Web
(Bedst på Nettet), a danish web assessment award, who is required to
inject code in the html of their site.

The module provides a simple way to embed the custom code required for
The top of the web (Bedst på nettet) (http://www.bedstpaanettet.dk/).

 * Automates injection of the embedded scripts in the right places.
 * Simplifies setup, just enter the key.
 * Shows you how to find the right key with screenshot.

INSTALLATION
------------

Download to sites/all/modules or the module directory of your choice
and enable in the Drupal module page.

CONFIGURATION
-------------

Access the configuration page at admin/config/system/bedstpaanettet (Or 
use the link in the Configuration page. Here you can unfold the instructions
if needed, and enter the key as found on The Top of the Web website.

KNOWN ISSUES
------------

No known error conditions.

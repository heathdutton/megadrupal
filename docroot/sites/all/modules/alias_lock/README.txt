SUMMARY: Allows locking of aliases so they won't be lost in the bulk delete op.


INSTALLATION:
* Download and unzip this module into your modules directory.
* Goto Administer > Site Building > Modules and enable this module.


USAGE:
* You can lock/unlock an alias on node edit forms; or in bulk at /admin/content

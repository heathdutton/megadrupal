1. ABOUT
========

This module allows you to set preferred input format for each field of any entity type (nodes, comments and etc).
As the site administrator you can choose global (for all users) preferred format for field
or select preferred format for field according to user role.

2. INSTALLATION
===============

Just install as usual, see http://drupal.org/node/70151 for further information.
Configure preferred format for your field in any entity.

Contact me with any questions.

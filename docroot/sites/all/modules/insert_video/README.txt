Extends Insert module functionality to Video fields provided by Media module.

Has been tested with Media:Youtube 7.x-2.x


INSTALLATION

- Enable the insert_video module.
- Add a file field to your content type and set the widget to Media selector.
- Edit the field and enable 'Insert' support. Be sure to select one or more
  formatters!
- Create content, add a video, insert the video.

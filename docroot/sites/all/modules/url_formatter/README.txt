DESCRIPTION
____________________

This module implements formatter that can change url formatt into another one,
like image, video etc.


INSTALLATION
____________________

To install this module, do the following:

1. Extract the tar ball that you downloaded from Drupal.org.

2. Upload the url_formatter directory and all its contents to your modules
   directory.

3. Visit admin/modules and enable the "Url formatter" module.


API
____________________

One can find some API in url_formatter.api.php with description.

Introduction
------------
webform_userpoints is a simple module to allow the management and processing of userpoints when a webform has been completed.

Dependencies
------------
Userpoints
Webform - Utilises changes made in v3.x of the webform module.

Installation instructions
-------------------------
1) Copy this module to you modules directory, typically sites/all/modules.
2) visit http://<yoursite>/admin/build/modules
3) Enable the Webform User Points Module

This will add a new configuration option for userpoints allocation to each individual web form settings page.

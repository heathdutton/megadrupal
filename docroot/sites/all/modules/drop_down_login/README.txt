INTRODUCTION
------------
This module provide a drop down block for user login,
User can display this block in any region for all / specified pages.
This is a tiny project, Login block display on login button toggle,
if javascript is disabled clicking on the button will redirect user
to user login page. 
 
REQUIREMENT
------------
Drupal 7.x

INSTALLATION
------------
1. Copy the entire drop_down_login directory the Drupal 
sites/all/modules/custom directory.

2. Login as an administrator. Enable the module in the 
"Administer" -> "Modules".

CONFIGURATION
-------------

1. Add the new block "Drop down Login" to the required 
region via "Administer" -> "Structure" -> "Blocks".

Description
-----------
This is a very simple add-in to the webform 
http://drupal.org/project/webform module.
It allows the user to add a user defined class to any element on their forms. 
Thus making webforms very easy to theme.

Use
---
To add a class to a webform component.
1) On your webform click webform.
2) Find the component on the list and click edit.
3) Find the fieldset titled Classes.
4) Under Class add classes for this component seperated by spaces.

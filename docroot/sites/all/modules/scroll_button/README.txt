INTRODUCTION
------------
This module has ability to scroll page on the top and (or) to the bottom.
It's a block that you can place everywhere, use all features are provided by
block core module and also additional features via this module.

 * For a full description of the module, visit the project page:
   https://www.drupal.org/project/scroll_button

 * To submit bug reports and feature suggestions, or to track changes:
   https://www.drupal.org/project/issues/2450807

REQUIREMENTS
------------
This module hasn't any requirements.

INSTALLATION
------------
 * Install as you would normally install a contributed Drupal module. See:
   https://drupal.org/documentation/install/modules-themes/modules-7
   for further information.

CONFIGURATION
-------------
 * Customize the module settings in Structure » Blocks » Scroll Button
   » configure. Also you can get the module settings via configuration link
   of installed "Scroll button" module.

-- SUMMARY --

The Bongo Panes module provides a Ctools content type for displaying a table of
Bongo Predictions.


-- REQUIREMENTS --

* Bongo
* Ctools - http://drupal.org/project/ctools


-- INSTALLATION --

* Install as usual, see
  http://drupal.org/documentation/install/modules-themes/modules-7


-- USAGE --

* If using Panels, when adding content to a panel, a Bongo Predictions pane is
  now available in the Bongo category.


-- KNOWN ISSUES --

When using the Panels In Place Editor (IPE), the predictions pane edit form
fails to load. - https://drupal.org/node/2075275

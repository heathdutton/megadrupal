Sub modules
============================

1. Simple Contest Repeat
	Provides the facility to create automated or repeated contests.

2. Simple Contest ubercart Coupon
	Simple Contest wintegration with Ubercart Coupon.
	
3. Simple Contest Ubercart Gift Certificate
	Simple Contest integration with Ubercart gift certificate.

4. Simple Contest rules
	Simple Contest integration with Rules.

5. Simple Contest Userpoints
	Simple Contest integration with Userpoints.

6. Simple Contest Commerce Coupon (D7)
	Simple Contest integration with Commerce Coupon.
	
Dependencies
============================
	Token : http://drupal.org/project/token [E-mail tokens Integration]
	Views : http://drupal.org/project/views [Contest winners selection]
	Date : http://drupal.org/project/date_repeat [Contest Date]

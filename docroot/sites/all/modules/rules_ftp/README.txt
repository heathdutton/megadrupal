
-- SUMMARY --

This module integrates FTP commands into Drupal via the Rules module allowing
events to trigger the sending of files to an FTP server. Example rules are
provided.

* Current supported commands:
  - put

For a full description of the module, visit the project page:
  http://drupal.org/project/rules_ftp

To submit bug reports and feature suggestions, or to track changes:
  http://drupal.org/project/issues/rules_ftp


-- REQUIREMENTS --

* Rules
  https://drupal.org/project/rules


-- INSTALLATION --

* Install as usual, see
  http://drupal.org/documentation/install/modules-themes/modules-7


-- CONFIGURATION --

Provided actions can be configured via the Rules module (see example rules).

The GoToWebinar module provides an interface to register users by role to
scheduled Webinars.

GoToWebinar was written and is maintained by Stuart Clark (deciphered) of
Realityloop Pty Ltd.
- http://www.realityloop.com

GoToWebinar was initially developed for Business Spectator Pty Ltd.
- http://www.businessspectator.com.au


Features
--------------------------------------------------------------------------------

- User entity field to Registrant field mapping interface.
- Register user by role.

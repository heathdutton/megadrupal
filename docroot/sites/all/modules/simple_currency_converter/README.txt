Simple Currency Converter Module
==============================================================================


Introduction
------------

Convert all the prices on your site to any currency for better user experience.



Usage
-----

As an example add the following mark to you page.

<a class="scc_modal_window_trigger" href="/" >Open Currency Selector</a>

Convert from USD: <span class="scc_price" data-amount="100" data-currency-code="AUS">&nbsp;</span><br/>
Convert from EUR: <span class="scc_price" data-amount="100" data-currency-code="EUR">&nbsp;</span><br/>
Convert from GBP: <span class="scc_price" data-amount="100" data-currency-code="GBP">&nbsp;</span><br/>



Features
--------

1. Covert prices on the page to any currency.

2. Multiple currency feeds can be used:
    • Google Finance
    • European Central Bank
    • Westpac Bank (NZ)

3. Currency feeds are plugable so you can write your own.

4. Admin notification when conversion check fails.

5. Flexible configuration



Basic Installation & Configuration
------------------

1. Download and enable the module.

2. Go to Administration > Configuration > Regional and language > Simple Currency Converter (admin/config/regional/simplecurrencyconverter).

3. Click on Manage Display for the relevant content type. (admin/structure/types/manage/page/display)5. Click on the settings gear to define your settings.

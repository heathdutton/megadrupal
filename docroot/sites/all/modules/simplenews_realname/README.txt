
-- SUMMARY --

The Simplenews RealName module allows you to send personalized newsletters
with the Simplenews module by collecting the name of the subscribers, which will
be available in the messages as token.

For a full description of the module, visit the project page:
  http://drupal.org/project/simplenews_realname

To submit bug reports and feature suggestions, or to track changes:
  http://drupal.org/project/issues/simplenews_realname


-- REQUIREMENTS --

* Simplenews module - http://drupal.org/project/simplenews


-- INSTALLATION --

* Install as usual, see http://drupal.org/node/70151 for further information.


-- RECOMMENDED --

* RealName module - http://drupal.org/project/realname

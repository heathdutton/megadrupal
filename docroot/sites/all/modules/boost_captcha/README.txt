Description
-----------
Allows static caching (boost or varnish or other static page caching mechanism) of pages with forms with captcha.

Features
--------
1. Dynamically generate the captcha to be used for submitting the form.

Configuration
-------------
1. Visit 'administer >> site configuration >> boost captcha'
2. You can specify the specific forms which should be handled by the module. 

Installation
------------
1. Extract the tar.gz into your 'modules' or directory.
2. Enable the module at 'administer >> site building >> modules'.
3. Configure options in admin/settings/boost-captcha

Uninstallation
--------------
1. Disable the module.
2. Uninstall the module

Description
----------

Provides a field type Mini Panel reference which creates a relationship to a 
Mini Panel display and allows the mini panel to be displayed as the content 
of the field.

Installation
------------

 * Mini Panel Reference depends on the panels module, download and install it 
   from http://drupal.org/project/panels
 * Copy the whole mini_panel_reference directory to your modules directory
   (e.g. DRUPAL_ROOT/sites/all/modules) and activate the 
	 Mini Panel Reference module.
 * Add a field of type "mini panel reference" to your entity

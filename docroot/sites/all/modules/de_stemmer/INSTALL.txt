
Installation
------------

1. Copy the de_stemmer folder to the Drupal modules/ folder.

2. Go to 'administer > Site building > modules' and enable this module (German stemmer)

3. Go to 'administer > Site configuration > search settings' and click 'Re-index site'.

4. Optional: Apply the patch for the search module included in this package 
   to mark keywords found in the text

INTRODUCTION
------------
Maintainer: Azri solutions (http://drupal.org/user/115278)

User can input the body weight and height and the module calculates the Body Mass Index of the user and will show it. 

INSTALLATION
------------
1. Copy the module bmi folder to modules directory.
2. At admin/build/modules enable the bmi module.
3. Enable permissions at admin/user/permissions.
4. Configure the module at admin/settings/bmi_settings.
5. Once module is enabled, a bmi block will be visible in admin/build/blocks.
6. Enable the block in required region and you will see a form.

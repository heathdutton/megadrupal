INTRODUCTION
------------

This module provides a new context condition. When viewing an user profile
(view mode full), It allows you to check if the profile has one of the roles
selected.

Use case: displaying different blocks on user profiles based on their roles.

 * For a full description of the module, visit the project page:
   https://www.drupal.org/project/context_profile_role
 * To submit bug reports and feature suggestions, or to track changes:
   https://www.drupal.org/project/issues/2413605


REQUIREMENTS

This module requires the following module:
 * Context (https://www.drupal.org/project/context)

-------------
Profile Alert
-------------

This module alert users to fill out empty fields in their user profiles.

Getting started
---------------

1. Enable module on Drupal module page or using Drush command.
2. Go to module administration page on path admin/config/people/profile-alert
3. Set up some field that you want to alert on your users. (you have to create
   some before setting this)
4. Assign block Profile Alert to some visible region in your Drupal theme.
5. You can configure your newly placed block with some descriptional text.

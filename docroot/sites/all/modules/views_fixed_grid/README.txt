Welcome to Views Fixed Grid

Installing Views Fixed Grid:

Place the entirety of this directory in sites/all/modules/views

Navigate to administer >> build >> modules. Enable Views Fixed Grid.

If you are having problems installing the module please see:
http://drupal.org/documentation/install/modules-themes/modules-5-6

This module depends on Views and has no meaning without Views.

If you installed the module correctly you should see a new option called
'Views Fixed Grid' on Views UI (Style settings >> Styles).

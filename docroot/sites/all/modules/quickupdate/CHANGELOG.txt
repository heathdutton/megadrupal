
Quick update 7.x-1.1, 2013-10-20
--------------------------------
- Added drush command "drush qup-list-md" to list current missing dependency
  projects.
- Added drush command "drush qup-dl" to download projects and their dependency
  projects.
- Changed drush command from "drush qup-md" to "drush qup-dl-md". It will
  download all missing dependency projects.

- Altered admin update form at /admin/reports/updates/update to search and
  download projects.
- Added "drush qup-md" to download all missing dependency projects.

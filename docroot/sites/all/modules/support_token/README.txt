
-- SUMMARY --

Support Token provides tokens support for tickets created with the Support
module.

To submit bug reports and feature suggestions, or to track changes:
  http://drupal.org/project/issues/support_token


-- REQUIREMENTS --

* Support

* Token


-- INSTALLATION --

* Install as usual, see http://drupal.org/node/70151 for further information.


-- CONFIGURATION --

* There is no configuration. Once installed the tokens are available.

-- TROUBLESHOOTING --

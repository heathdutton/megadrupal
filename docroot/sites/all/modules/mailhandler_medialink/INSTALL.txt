INSTALLATION:                                                       ************

There are no special installation instructions for this module.
See: http://drupal.org/documentation/install/modules-themes

The module does however have dependencies on
ctools
media
media_internet
mailhandler

ENABLING FUNCTIONALITY:                                             ************

In your feeds importer under Mailhandler IMAP stream parser settings:
Select Mail Media Link Processor from the Command plugins select list.

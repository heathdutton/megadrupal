Autofill Forms
================================

Description
-----------

Autofill forms module provides automatically generate form field default values.
This module will not overwrite the field default values.

Features
--------
The site administrator can able to-

1. Automatically generate form field default values.

2. Enable/Disable which content type need to be generate default value.

3. Enable/Disable form field type.

4. Configure how many words generate text/teaxarea fields.

5. Will not overwrite field default value.

6. This module support node form only.



Dependencies
------------
Drupal 7 latest version.


Installation
------------
Download and install the Autofill forms module as normal.

Install modules on "sites/all/modules/contrib/" folder.

Configuration
-------------
1. Go to config page (admin/config/content/autofill/configure).

2. Configure the form field (which field need to be autofill default value).

3. Configure content type (which content type need to be autofill default value).

4. Configure number of words for text/textarea field.

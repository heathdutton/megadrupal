-- INTRODUCTION --

The India Commerce PayU module provide a payment method for Drupal
commerce to Interact with PayU payment gateway.


-- REQUIREMENTS --

Drupal Commerce(https://www.drupal.org/project/commerce)


-- INSTALLATION --

* Install as usual, see http://drupal.org/node/895232 for further information.
* Goto admin/commerce/config/payment-methods and enable a rule for PayU.
* Edit Action and Configure the Key & Salt.

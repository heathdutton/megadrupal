=== LINKEDIN GROUP POSTS ===


Installation/initial configuration

   1) Enable OAuth, LinkedIn and its submodule Linkedin profile integration
      from modules page.

   2) Go to LinkedIn Integration config page. admin/config/services/linkedin

   3) Set Library path as 'sites/all/modules/oauth/lib/OAuth.php'

   4) Enter API Key and Secret Key which you obtained by requesting.

   5) Enable LinkedIn group posts module. Edit your account 'user/1/edit' and 
      click on LinkedIn tab to authorize your drupal account with linkedin.
  
   6) Go back to admin/config/services/linkedin page and save your
      configurations for 'LinkedIn Group Integration'

   7) Open Block page and there you can find the Group posts blocks.

INSTALLATION
------------

Decompress the whatcounts-7.x-n.m.tar.gz file into your Drupal modules
directory (usually sites/all/modules, see http://drupal.org/node/176044 for
more information).

1. Enable the module: Administration > Modules (admin/modules)

2. Adjust user permissions: Administration > People > Permissions (admin/people/permissions)

3. Configure the settings for this module: Administration > Configuration > People > WhatCounts (admin/config/people/whatcounts)

4. Enable the blocks: Administration > Structure > Blocks (admin/structure/block)

-- SUMMARY --

Provides possibility to loads any pane in panels via ajax.


-- REQUIREMENTS --

Ctools http://drupal.org/project/ctools
Panels http://drupal.org/project/panels


-- INSTALLATION --

* Install as usual, see http://drupal.org/node/70151 for further information.


-- HOWTO --

1. Add any pane to panel
2. Click on edit (gear icon) and click change Caching.
3. Select AJAX pane caching method
4. Next and Save

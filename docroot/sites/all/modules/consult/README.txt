INTRODUCTION
------------
With the Consult module, developers and site builders are able to create
questionnaires that present site users with a set of possible results based on
answers to a set of defined questions.

GETTING STARTED
---------------
1. Enabled the module including the Entity API and backbone library
   dependencies.
2. Visit admin/structure/consult and click the 'Add interview' link.
3. Create a label and machine name for the new interview and select Save
   consultation.
4. Add questions and results using the action links on the interview page.

DESCRIPTION
-----------
Gives site builders the ability to disable zones in the "State/Province" 
dropdown on the checkout page. Zones in the billing and shipping pane can be 
disabled separately (to allow someone to buy a product from Hawaii address 
without shipping there for example).

Tested with Ubercart 6.x-2.4 only

INSTALLATION
------------
Install like a standard contributed module: http://drupal.org/node/70151

CONFIGURATION
-------------
Disable the zones you don't want to ship to, or have people buy from at: 
Administer > Store administration > Configuration > Disable Zones

Description
-----------
This module adds a pager for the webform module
found at: (http://http://drupal.org/project/webform).
The pager shows how many pages a user has left in a multi-page webform.
When hovering the page name a page description can be shown through jQuery.

Requirements
------------
Drupal 7.x
Webform module (http://http://drupal.org/project/webform)

Installation
------------
1. Copy the entire webform_pager directory the Drupal 
  sites/all/modules directory.

2. Login as an administrator. Enable the module in the "Administer" -> "Modules"

3. (Optional) Edit the settings under "Administer" -> "Configuration" ->
   "Content authoring" -> "Webform pager settings"

4. Create a webform node at node/add/webform.

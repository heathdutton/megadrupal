This module updates the external URLs present in the body content
and change them to be opened in the new tab.


Installation
-------------

* Copy the whole "update_external_links" directory to your modules
  directory - "sites/all/modules"
* Enable the module.
* Set your desired bundle wise permissions at - "admin/people/permissions".


Usage
------
   
 * Go to "/admin/config/system/update-external-links".
 * Select the required options.

Permissions available
---------------------
* Access Update external links Form

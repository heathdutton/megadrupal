================================================================================

With "Explorer 8 Mode" can be defined as the Internet Explorer 8 displays eight
pages of an Drupal installation.

Requirements
--------------------------------------------------------------------------------
This module is written for Drupal 7.0+.

Installation
--------------------------------------------------------------------------------
Method 1
Copy the Explorer 8 Mode module folder to your module directory and then enable
on the admin modules page.

Method 2
Use the GUI to install modules (admin/modules/install) and then enable on the
on the admin modules page.

Administration
--------------------------------------------------------------------------------
Administer the module in Administer -> Configuration -> System;
admin/config/system/explorer8_mode

Which mode to use for Explorer 7?

In order to use the Explorer 8 as the Explorer 7, you choose the mode
"Emulate IE 7".


DocRaptor PDF Library
=====

Installation
============

- Install this module in sites/all/modules
- Get php wrapper here https://github.com/apmsooner/docraptor/archive/master.zip
- unzip wrapper to sites/all/libraries/docraptor.
- Path should look like this: /sites/all/libraries/docraptor/docraptor.php
- Enter your API key here: admin/config/user-interface/print/pdf/docraptor


Docraptor runs on the powerful princexml platform.
Everything you can do is documented here: http://www.princexml.com/doc/9.0/.

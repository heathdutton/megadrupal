-- SUMMARY --

Patterns Client is a module that extends Patterns allowing the exchange (push/pull) 
of pattern files from any site running a Patterns Server.
It relies on D2D to exchange the information in a distributed and secure way.

-- REQUIREMENTS --

* D2D: https://drupal.org/sandbox/shakty/1889370


-- INSTALLATION --

* Install as usual, see http://drupal.org/node/70151 for further information.

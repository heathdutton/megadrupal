
Module: GetClicky
Authors: gloscon, rsvelko (after Sep 2009), pcambra (Drupal 7 version)


Description
===========
Adds the GetClicky tracking system to your website.

Requirements
============

* GetClicky user account


Installation
============
* Copy the 'getclicky' module directory in to your Drupal
modules directory as usual.


Usage
=====
In the settings page enter your GetClicky ID (a number used to build your affiliate link).

All pages will now have the required JavaScript added to the
HTML footer can confirm this by viewing the page source from
your browser.

'admin/' pages are automatically ignored by Getclicky.

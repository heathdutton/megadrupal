INSTALLATION INSTRUCTIONS
-------------------------
1.  Copy the files included in the tarball into a directory named "tumblr_connect" in
    your Drupal sites/all/modules/ directory.
2.  Login as site administrator.
4.  Enable the Tumblr Connect module on the Modules -> LIST page.
5.  Fill in required settings on the Modules -> Tumblr Connect -> Configure page.
6.  Optional: Recommended if your Tumblr account already has followers.
    Enable the Tumblr Follower block on the Structure -> Blocks page. Run Modules ->
    Tumblr Connect -> Configure -> Run Cron Task
7.  Enjoy.


Tumblr Connect module for Drupal 7.x.
This module exports Drupal content to Tumblr..

REQUIREMENTS
------------
* Access to an account on Tumblr.
  The Tumblr service can be found here:
  http://www.tumblr.com/
* PHP 5.1.0 or greater.
* The following PHP extension needs to be installed: cURL
* The following Drupal module needs to installed: simplehtmldom API
  The simplehtmldom API module can be found here:
  http://drupal.org/project/simplehtmldom
  

INSTALL INSTRUCTIONS
--------------------
See INSTALL.txt

NOTES
-----
This module sends Drupal content to your Tumblr account.  Therefore, you need
to have access to a Tumblr account for this module to work.

INTRODUCTION
------------
Media: Prezi integrates the Prezi presentation sharing service with the Media
module to allow users to add and manage Prezi as they would any other piece of
media.

Internet media can be added on the Web tab of the Add file page (file/add/web).
With Media: Prezi enabled, users can add a Prezi presentation by entering its
URL or embed code.

 * For a full description of the module, visit the project page:
   https://www.drupal.org/sandbox/ogilbert/2288159
 * To submit bug reports and feature suggestions, or to track changes:
   https://www.drupal.org/project/issues/2288159


REQUIREMENTS
------------
This module requires the following modules:
 * Media 7.x-2.x (https://www.drupal.org/project/media)
 * Media Internet (A submodule of the Media module)


INSTALLATION
------------
 * Install as you would normally install a contributed drupal module. See:
   https://drupal.org/documentation/install/modules-themes/modules-7
   for further information.


CONFIGURATION
-------------
The module has no menu or modifiable settings. There is no
configuration.  When enabled, the module will provide the file type
presentation like other file types (image, video...)

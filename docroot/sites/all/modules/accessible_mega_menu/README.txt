
-- SUMMARY --

This module provides integration with the Accessible Mega Menu jQuery plugin. An
Accessible Mega Menu block is created for each menu. Accessible Mega Menu is an
open source jQuery plugin provided by Adobe Accessibility
(https://github.com/adobe-accessibility).

For a full description of the module, visit the project page:
  http://drupal.org/project/accessible_mega_menu

To submit bug reports and feature suggestions, or to track changes:
  http://drupal.org/project/issues/accessible_mega_menu


-- REQUIREMENTS --

* jQuery Update
  https://drupal.org/project/jquery_update

* Libraries API (2.x)
  https://drupal.org/project/libraries

* Accessible Mega Menu jQuery Plugin
  https://github.com/adobe-accessibility/Accessible-Mega-Menu


-- INSTALLATION --

1 Download the Accessible Mega Menu jQuery plugin from github:
  https://github.com/adobe-accessibility/Accessible-Mega-Menu

2 Extract the Accessible Mega Menu jQuery plugin into a libraries directory
  (e.g., sites/all/libraries).

3 Rename the 'Accessible-Mega-Menu-master' directory to 'accessible_mega_menu'.

4 Install the Accessible Mega Menu module as usual, see
  http://drupal.org/documentation/install/modules-themes/modules-7

5 Verify proper installation of the library at admin/reports/status.


-- CONFIGURATION --

1 Configure the Default jQuery Version to be >= 1.10:
  admin/config/development/jquery_update

2 Place Accessible Mega Menu blocks in the desired regions.

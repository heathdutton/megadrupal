
-- SUMMARY --

The Commerce Product Reference Views Widget module adds a new widget to the Commerce Product Reference field type.
The new widget has the ability to accept a view as input source for the commerce product reference field type.

For a full description of the module, visit the project page:
  http://drupal.org/project/product_reference_view
To submit bug reports and feature suggestions, or to track changes:
  http://drupal.org/project/issues/product_reference_view


-- REQUIREMENTS --

* Views
* Commerce Product Reference
* Entity API
* Viewfield


-- INSTALLATION --

* Install as usual, see http://drupal.org/node/70151 for further information.

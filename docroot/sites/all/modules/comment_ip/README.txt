INTRODUCTION
------------

Current Maintainer: James Davidson <jamesdavidson87@gmail.com>

The module displays a commenters IP address on the comment overview page and
adds the additional option to delete and block the commenters IP from the
comment overview dropdown.

USAGE
-----

Once enabled, there will be a new column added to the comment overview page, and
a new option in the dropdown box. Selecting a one or many comments and selecting
"Delete the selected comments and block their IP's" from the dropdown will
delete the comments and add that commenters IP to the blocked IP list.

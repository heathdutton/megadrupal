DESCRIPTION
===========
Europabank (http://www.europabank.be) 
integration for the Drupal Commerce payment and checkout system.

INSTALLATION INSTRUCTIONS FOR COMMERCE EUROPABANK
=================================================
1. Create a test account at 
https://www.ebonline.be/test/home.jsp?link=register&lang=nl

2. Europabank wil mail you some test cards and login information.

3. Install the module as usual.

4. enable the payment method at 
www.example.com/admin/commerce/config/payment-methods

You can find a detailled guide on how to configure payment methods at
http://www.drupalcommerce.org/user-guide/payments

5. Configure the payment gateway with the settings received in your email from 
mpi@europabank.xxx 

6. Start handling payments.

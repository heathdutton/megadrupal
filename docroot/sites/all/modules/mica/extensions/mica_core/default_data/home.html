Mica is a powerful software application used to create web portals for epidemiological study consortia. Using Mica, consortia can build personalized web sites for publishing their research activities and their membership.

Studies

List of the consortium's studies. Each study is described in a same standard way that can be easily customized to reflect consortium specificities.

Dataset & Variables

Mica describes datasets, their scientific purposes and studies that contribute to each of them. Each dataset is described by a comprehensive data dictionary which gives details on each available variable.

Data Access

Mica presents your data access policy, manages data access requests from submission to approval and presents the data access committee members

Community

Mica offers out of the box different communication means such as forums, blogs, news and events announcement.

Mica includes many domain-specific features such as study catalogues, data dictionary browsers, on-line data access request forms, community tools (forums, events, news) and others. Moreover, Mica includes a powerful data search engine that allows authenticated researchers to perform distributed queries on the content of each individual study data collection hosted by the OBiBa Opal database software.

Mica is based on the content management software Drupal used by millions of websites worldwide. As such, a webmaster benefits from all the power and flexibility of Drupal, as well as the support of an extended developer community.

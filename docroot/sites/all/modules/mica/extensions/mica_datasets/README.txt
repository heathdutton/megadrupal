Mica Datasets Module


INSTALLATION
------------------------

Download prettify library:
http://code.google.com/p/google-code-prettify/downloads/detail?name=prettify-small-1-Jun-2011.tar.bz2

Extract it to
DRUPAL_ROOT/sites/all/libraries/prettify/
  - prettify.js
  - prettify.css
  - etc...

The library should then be found by the module.

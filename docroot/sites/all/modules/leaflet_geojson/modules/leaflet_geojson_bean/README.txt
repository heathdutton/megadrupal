
Leaflet GeoJSON Bean
-----------

This module provides a custom bean type for Leaflet Maps
based on Views GeoJSON page displays.

USAGE
-----------

Go to your.site/block/add to create Leaflet GeoJSON Bean blocks.

GeoJSON sources are exposed by the leaflet_geojson module.
For example, Views GeoJSON pages are automatically available.

Also, refer to the module specific documentation pages:
http://drupal.org/project/bean
http://drupal.org/project/leaflet
http://drupal.org/project/views_geojson

INTRODUCTION
------------

This module allows additional columns/data to be added to the node, comment,
and user tables in administration area. It provides a list of fields defined
in your drupal installation as well as a few custom fields to choose from.

HOW TO USE
----------

1. download and enable module
2. go to configuration page(/admin/config/extra_columns) to
   determine which form you want Extra Columns to be on
3. go to chosen form(e.g. /admin/content)
4. Click on the + header cell and choose which fields you want to display

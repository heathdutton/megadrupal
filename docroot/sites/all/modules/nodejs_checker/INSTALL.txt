 * @File
 *
 * Installation guide for Drupal nodejs_checker module.
 */
 
1. Install the nodejs integration module.

2. Copy / Paste the nodejs_checker.server.extension.js file from the
Nodejs livetracking module to the Nodejs integration module.

3. Activate the nodejs_checker module from your module panel.

4. Add the extension to your Nodejs integration module configuration file :
nodejs.config.js
=> extensions: ['nodejs_checker.server.extension.js'],

5. Restart your Nodejs server.

6. Activate the nodejs_checker block from your admin pannel. Enjoy.

INSTALL
--------------------------------------------------------------------------------
  1. Check if you enabled soap extension ( if you have not read NuSoap )
  2. Goto admin/store/settings/payment/edit/methods and set Saman Bank settings
  3. Use Saman Bank module !

Nusoap 
-------------------------------------------------------------------------------- 
if you have not nusoap you need to download it's php file and rename it to nusoap.php and copy it to saman bank module directory !

-- SUMMARY --

Alters the fivestar module to allow voting with half stars.


-- REQUIREMENTS --

* Drupal 7
* The fivestar module.


-- INSTALLATION --

* Install as usual, see http://drupal.org/node/70151 for further information.



-- CONFIGURATION --

* When editing the fivestar field instance settings there is a checkbox to 
  allow voting with halfstars
* For further help refer to: https://drupal.org/sandbox/dahousecat/2233333

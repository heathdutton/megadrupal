Mobile Switch Blocks

Control block visibility in mobile devices using Mobile Switch.

DEPENDENCIES
------------

  - Drupal 7.
  - The Mobile Switch (branch 2) module from drupal.org/project/mobile_switch.

INSTALL
-------

  1) Copy the Mobile Switch Blocks folder to the modules folder in your
     installation. Usually this is sites/all/modules.
     Or use the UI and install it via admin/modules/install.

ADMINISTER
----------

  Administer blocks under Administration -> Structure
  admin/structure/block

  Use the 'configure' link of the blocks and configure the vertical tab
  'Is mobile' in the 'Visibility settings' area.

LIMITATION
----------

  The module not works with a desktop browser using a mobile device
  user agent string.

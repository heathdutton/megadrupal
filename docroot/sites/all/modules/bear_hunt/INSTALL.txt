Dependencies
------------

Solr Server
-----
You will need to install and configure your Solr server on your server in order for this module to work. For information on installing and configuring a Solr server, refer to http://drupal.org/node/1053074.

Library
-----
You will need to download the Solr PHP client library from http://code.google.com/p/solr-php-client/downloads/list. This library can be placed at sites/all/libraries and must be present before enabling the Bear Hunt module.

Post-installation
-----
Upon installing the module, navigate to admin/config/search/search_api/server/solr/edit and configure the Solr host, Solr port, Solr path and any other default settings to meet your needs.


INSTALLATION
------------

1. Follow the usual module installation procedure (2).
2. Do not forget to install the dependency 'libraries api' module (3).
3. Visit the status page (4) to check barcode is installed correctly.
4. Visit the settings page (5) to review the global settings.
5. Make sure to put Barcode.php & arial.ttf (6) in 'regbarlibs' folder inside
   sites/all/libraries, so the path should be:
   e.g. sites/all/libraries/regbarlibs/Barcode.php

[2] http://drupal.org/documentation/install/modules-themes/modules-5-6
[3] http://drupal.org/project/libraries
[4] admin/logs/status
[5] admin/barcode/settings
[6] https://github.com/nmudgal/regbar_libs

USAGE
-----

This module integrates with Drupal registration process to provide advanced
functionality or generation of icards having either randomized barcodes or
barcode with unique field such a Drupal username which is provided using user
module.

UPGRADING FROM 6.x TO 7.x

The following describes how to upgrade multisite_search module from 6.x to 7.x.

1. Replace multisite_search/ module folder with 7.x files.

2. Run http://example.com/update.php .

3. Visit admin/config/search/settings and enable Multisite Search under "Active search modules".

4. Visit admin/config/development/performance and click "Clear all caches".

5. Run cron (link at admin/reports/status).

6. Repeat steps 2-5 for all sites.

Confirm by running a test at search/multisite_search .

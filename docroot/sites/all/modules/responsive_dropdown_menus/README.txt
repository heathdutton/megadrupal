This module provides a theme() function to provide tiered menu links for the
site's main menu. This module provides barebone functionality that provides
dropdown support for tiered menus and responsive styling. Beyond the main menu
the module exposes all menus as blocks handled via Responsive Dropdown Menus.

Installation:
Just enable the module, adapt CSS to your needs and enjoy your responsive menu.

About
* Viewports less than 740px menus will be hidden and have a menu toggle
  displayed, beyond 740px menus displayed normally.
* Menu toggle displays the menu's display title.
* Utilizes max-height to allow CSS animation implementation.
* Dropdown styling provides simply display and hide without positioning
  (allowing themers full control without overriding styles.)

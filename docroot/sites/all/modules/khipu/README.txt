/**
 * Khipu
 */

This module implements a payment method called Khipu from Chile and this is
only used in this country.

/**
 * Dependencies
 */

1) This module need the Khipu library (https://github.com/mnico/Khipu).
 Must be installed in the folder sites/all/libraries. It should be something
 like this sites/all/libraries/Khipu/src/Khipu.php
2) Libraries API (http://drupal.org/project/libraries)

/**
 * Additional modules
 */

This module has a submodule to implements the payment button like a field
to any entities with a mini system for orders.

There is also a module that allows to create different khipu accounts for each
system user.

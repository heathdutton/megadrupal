DESCRIPTION
-----------
This is a module to display a block for the user with links to the Moodle 
courses he is member of. The user names on Drupal must match the ones on Moodle.
You can select which Drupal roles have access to this block.

REQUIREMENTS
------------
Moodle Connector module.

NOTES
-----
Tested on Moodle versions 1.9 to 2.4.

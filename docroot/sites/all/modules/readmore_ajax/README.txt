INTRODUCTION
--------------------------

This is a simple module adds a link to a Read more 
the details in the teaser ajax.
This will in effect create a blog like on facebook
(no need to go back to the blog).


CONFIGURATION
--------------------------

This module requires no configuration. Just install and enjoy.

INFORMATION FOR DEVELOPERS
--------------------------

This module is designed for personal needs, and has two objectives: 
to make life easier for developers who collect site without delving into
writing code, and the second to show by example the possibility ajax in Drupal.

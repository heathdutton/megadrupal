PLANYO.COM ONLINE RESERVATION SYSTEM - Drupal 7 module

This module embeds the Planyo.com online reservation system. Before using it, you'll need to create an
 account at planyo.com. Please see http://www.planyo.com/drupal-reservation-system for more info.

Copy all files to a 'planyo' directory inside 'modules', then enable the module and in module's properties, set your planyo site ID (default settings use a demo site).

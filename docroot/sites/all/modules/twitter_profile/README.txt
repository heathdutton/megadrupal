About "Twitter Profile Block"
=============================
Twitter Profile block given an enhance functionality to show all your twitter profile popularity at one place, It also have various twitter option to configure such as tweets count, friends images and  Pre build 3 different theme preset styles. You may configure as per your requirement, Increase your followers & social network with twitter profile.


Installation
============
Add twitter profile to your modules dir (at "site/all/modules")
After you activated the module (at "admin/modules")



Configuration
=============
Go to "admin/structure/block/manage
Find Twitter profile Confiqure it
Enter Twitter Username
Chose other setting as per your requirement
Select region example:Sidebar & Save

DESCRIPTION:
------------
This module provides comment delete according content type.

-- REQUIREMENTS --

comment module

INSTALLATION:
-------------
1. Place the entire comment_delete directory into your Drupal sites/all/modules/
   directory.

2. Enable the comment_delete module by navigating to:

     administer > modules

3. After enable module you will find new menu(Comment Delete)
   on content listing page.
    
    administer / content

INTRODUCTION
------------
Provides the ability to load the contents of a field via Ajax.

Maintainer: Daniel Braksator (http://drupal.org/user/134005)
Project page: http://drupal.org/project/peekaboo.


INSTALLATION
------------
Install and enable the Peekaboo; Ajax field formatter module.
For detailed instructions on installing contributed modules see:
http://drupal.org/documentation/install/modules-themes/modules-7


CONFIGURATION
-------------
When you edit a field there is a tab labelled "Manage display".  On that page
next to your field's display summary there is a little settings button that 
usually looks like a cog.  Clicking the button will pull up a form which 
includes the Peekaboo settings.

For general instructions on using the Field UI see: 
http://drupal.org/documentation/modules/field-ui

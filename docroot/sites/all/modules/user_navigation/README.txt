
-- SUMMARY --

The User Navigation module creates a simple navigation menu for all the users. 
It creates a new menu called "User Navigation Menu" where in the admin can add 
or remove the links to be shown.

For a full description of the module, visit the project page:
 https://drupal.org/sandbox/mudrasamey/2211983

-- REQUIREMENTS --

None.


-- INSTALLATION --

* Install as usual, see http://drupal.org/node/70151 for further information.

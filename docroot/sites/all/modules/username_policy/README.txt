
-- Summary -- 

This module allows admins to create policies for username.
Every user has a unique user name generated by the username policy module


-- Features -- 

Able to set usernames according to pattern set at username policy configuration.


-- Installation --

* The instalation of this module is like other Drupal modules.

1. Install This module (unpacking it to your Drupal
/sites/all/modules directory if you're installing by hand).

2. Enable this module in Admin menu > Site building > Modules.

3. Set permitions admin > people > permissions #access username policy control

-- Configuration --

* Configure settings at admin/config/people/username_policy

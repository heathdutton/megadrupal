This module includes some utilities related to ezmlm.
It has a block and a page with a form for subscribing for ezmlm lists.
It can be configured to allow new registrations to join lists.


Requirements
------------

This module requires Drupal 7.x

Installation
------------

1. Copy the ezmlm.module to the Drupal modules/directory. Drupal should
   automatically detect it.

2. Go to the admin menu and enable the ezmlm module

3. Go to the admin -> ezmlm lists menu and add / modify / delete lists.


Future Improvements Planned

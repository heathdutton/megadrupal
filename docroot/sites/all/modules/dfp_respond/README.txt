Module: DFP Respond

Description
===========
DFP Respond integration see https://azullo.desk.com

Installation
============
Copy the module directory in to your Drupal:
/sites/all/modules directory as usual.

Configuration
=============
visit admin/structure/dfp_ads/respond
or via menus Structure > DFP Ad Tags > DFP Respond

Create a new context and choose the reaction DFP Respond
Select which DFP Ad Tag you want to use and provide the selectors.
For more information on selectors, see the Documentation below.

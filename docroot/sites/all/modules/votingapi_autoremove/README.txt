Voting API Autoremove

Automatically removes orphaned votes when deleting content. Nodes and comments
are currently supported.


CONFIGURE
---------
- Enable and enjoy. No configuration required!

- Toggle the Autoremove checkbox on the VotingAPI configuration page to enable
and disable functionality.

- To delete existing orphaned votes, simply click the appropriate button on the
VotingAPI configuration page.

/**
  * Инструкция по установке
  */
  
1. Разархивируйте модуль в папку /sites/all/modules
2. Включите его на странице управления модулями admin/build/modules
3. Запустите выполнение регулярных процедур (cron)
4. Поставьте блок Информер Валют в любой удобный для вас регион
5. Настроить блок можно на странице admin/settings/currency_converter  


/**
  * Инструкция по эксплуатации
  */
  
Для работы модуля требуется запуск крона. Настройте его запуск примерно на 1 раз в сутки - и модуль вам будет выдавать всегда
свежие курсы валют

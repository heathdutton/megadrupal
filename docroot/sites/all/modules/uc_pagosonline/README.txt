DESCRIPTION:
------------
PagosOnline module integrates PagosOnline, a colombian merchant account, with
Ubercart Payment module.

INSTALLATION:
-------------
1. Extract the tar.gz into your 'modules' or directory.
2. Enable the module at 'administer >> site building >> modules'.

CONFIGURATION:
--------------
1. Visit 'Administer >> Store administration >> Configuration >> Payment Settings'
1.1. Visit 'Payment methods' to set up PagosOnline method.
1.2. Visit 'Payment gateways' to set up PagosOnline gateway.

UNINSTALLTION:
--------------
1. Disable the module.
2. Uninstall the module, which will delete the module variables.

DESCRIPTION:

This module enables libraries to integrate their catalog into Drupal 
allowing importing records, make advanced searches with faceted 
results, circulation task etc ...

This module is intended to work with any ILS by using connectors.
Everyone can make its own by creating a php file.

For a full description of the module, visit the project page:
  http://drupal.org/sandbox/AlexArnaud/1570572

The entire documentation is available (work in progress):
  http://drupal.org/node/1677586

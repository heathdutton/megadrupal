The purpose of this module is to provide the admin with option to add meta tag description for the websites that has multi-domain structure, this module requires Domain Access and Meta tags modules

Installation:
1- Download the Domains meta tag module
2- untar the previously downloaded file in sites/all/modules
3- enable the module from the admin interface in admin/modules
4- to add a new meta tag for a specific domain go to admin/config/search/metatags


Extended Search Admin
---------------------

Replace the default content admin overview with an extended search page.

The module uses the search api and the extended search page (see below).


Test
----

I've done test against solr index.


Requirement
-----------

Module search_api : http://drupal.org/project/search_api 
Module ext_search_page

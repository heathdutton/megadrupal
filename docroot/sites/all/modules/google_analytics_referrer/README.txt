Google Analytics Referrer is a scalable, lightweight system for gathering referrers detected by Google Analytics.

Installation, demo and tips are at http://vacilando.org/en/article/google-analytics-referrer
Drupal project page is at http://drupal.org/project/issues/google_analytics_referrer
If you encounter a problem or need support, see http://drupal.org/project/issues/google_analytics_referrer

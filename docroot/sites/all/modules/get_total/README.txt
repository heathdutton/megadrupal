GET TOTAL NUMBER OF CONTENT
=======================================
This module provides total number of content in a particular content type,It
will also shows how much content are published and unpublished and there name.
It will also show node count in /admin/content page.

Branch: 7.x-1.0

To enable this module navigate to

administer >> build >> modules. Enable Get Total.

To check number of number of content and there status please fallow these.

Home >> Administration >> Configuration >> Content authoring >> Get Total

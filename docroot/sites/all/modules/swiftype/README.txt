INTRODUCTION
------------
The Swiftype integration module adds new block with a search field on your
site. This module uses Swiftype Site Search Service. Swiftype Site Search
Service provides a number of features like:
  - custom result ranking,
  - autocomplete,
  - synonyms,
  - facets,
  - real-time indexing.

 * For a full description of the module, visit the project page:
   https://www.drupal.org/sandbox/dmitry-evdokimov/2539904

 * To submit bug reports and feature suggestions, or to track changes:
   https://www.drupal.org/project/issues/2539904

REQUIREMENTS
------------
No special requirements.

INSTALLATION
------------
Install as you would normally install a contributed Drupal module. See:
https://drupal.org/documentation/install/modules-themes/modules-7 for further
information.

CONFIGURATION
-------------
 * You need to configure new search engine on your Swiftype dashboard to
 get an install key. Check official Swiftype video tutorial to get more
 information about Swiftype Dashboard here: https://youtu.be/Un3tdQe5Xyg

 * You need to enter your Swiftype install key on the module's config page here:
 http://<your domain>/admin/config/search/swiftype

 * You also need to choose the region where Swiftype search block should be
 shown on your site here: http://<your domain>/admin/structure/block

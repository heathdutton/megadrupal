Installation
------------

1. Copy the scripturefilter.module to the Drupal sites/all/modules/ directory.

2. Go to administer > configuration > modules, and enable the
   scripturefilter.module

3. Go to administer > configuration > content authoring > text formats 
   and click "configure" against any text formats that you want to use
   this module. 
   You then need to (i) tick the box against Enabled filters > Scripture Filter,
   (ii) (optionally) choose a Default Bible translation under (Filter settings)
   (iii) (optionally) rearrange the "Filter processing order" to avoid any
    conflicts with other filters.

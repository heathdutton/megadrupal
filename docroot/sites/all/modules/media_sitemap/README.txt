Introduction
============
This module is used for creating sitemap for google. 
This module depends on the media module and fetch all the
image entries of media and create a proper and valid sitemap for
Google. 

To use 
-----------
1. Navigate to configuration  >> MEDIA >> Media Sitemap
URL : (admin/config/media/media_sitemap/list)
Note : If this option doesnt open then clear the cache and
re-open the URL.

2. To Generate the Sitemap for google Clcick on the 
"Add a new media sitemap" link.
3. This will generate the sitemap of all the images of media module.

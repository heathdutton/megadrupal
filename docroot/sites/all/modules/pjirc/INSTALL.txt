------------
INSTALLATION
------------

1) Copy pjirc module folder in your sites/all or sites/yoursite folder.

2) OPTIONAL: Upload a background image to pjirc/pjirc/background.gif.

3) Now you need to log in to your site and enable the new pjirc.module.
    (Goto :: administer -> modules :: then check 'pjirc'
        then push 'Save settings')


-------------
CONFIGURATION
-------------

4) Setup the server, room and guest name.
    (Goto :: administer -> settings -> pjirc)

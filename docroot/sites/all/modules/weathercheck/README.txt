INTRODUCTION
------------
Adds the Weather Check block to your website.

Weather Check is a simple module that lets the user to create a block
to display the weather condition at a particular location.
Default location is set from the administrator configuration page.
The anonymous user can move the location pin across the Google Map
to view the weather condition at that particular location.
However the administrator has the control to set what type of 
parameters to be displayed in the block.

Administrator can choose from any of these available parameters:
1. City Name
2. Country Code
3. Temperature
4. Minimum temperature
5. Maximum temperature
6. Cloud Description
7. Pressure
8. Humidity
9. Visibility level
10.Wind speed
11.Wind degree

This module uses:
1. AngularJS
2. Google Maps
3. OpenWatherMap API


INSTALLATION
------------
 * Install as you would normally install a contributed Drupal module. 	 See:
   https://drupal.org/documentation/install/modules-themes/modules-7
   for further information.


CONFIGURATION
-------------
In the configuration page enter latitude and longitude of 
the desired location or select the location from the map. 
Check the required parameters.

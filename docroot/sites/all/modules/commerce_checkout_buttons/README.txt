Commerce Checkout Buttons module

Use shorter button values on Drupal Commerce checkout pages.

Dependencies
============

Drupal 7.
Commerce Checkout module
Commerce Order module

Install and Configure
=====================

1) Copy the Commerce Checkout Buttons folder to the modules folder in your
   installation.
   Usually this is sites/all/modules.
   Or use the UI and install it via admin/modules/install.

2) No configuration or administration options available.

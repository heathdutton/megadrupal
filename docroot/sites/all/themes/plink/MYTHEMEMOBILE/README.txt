INSTALLATION INSTRUCTIONS
-------------------------------------------------------------------------------------

This theme is intended for use with mobile devices and should only be enabled for mobile. Administration capabilities are limited. It is reccommended that you set up mobile switching using a module such as mobile tools. 







CONTRIBUTED MODULES
-------------------------------------------------------------------------------------
The following contributed modules extend the functionality of the plink theme and you
should download and install them. They are not mandatory but highly recommended.

-----1.  Before installing Conch theme you need to download and install the jQuery
 Cycle plugin and SuperSleight. The jQuery
 Cycle plugin can be downloaded from:
    http://malsup.com/jquery/cycle/download.html and SuperSleight plugin can download from: http://allinthehead.com/code/sleight/supersleight.plugin.js.

 2. Place the jQuery Cycle plugin and SuperSleight plugin files in the [theme_directory]/conch/js directory and rename
 the file to supersleight.js or you can go to http://www.symphonythemes.com to download full version.

 See http://www.symphonythemes.com/sites/default/files/SymphonyTheme_Document_v0.1.1.pdf for more information on how to use Conch theme.

Installation:

  1. Download Conch from http://www.symphonythemes.com

  2. Unpack the downloaded file, take the entire alphone folder and place it in your
     Drupal installation under one of the following locations:
       sites/all/themes
         making it available to the default Drupal site and to all Drupal sites
         in a multi-site configuration
       sites/default/themes
         making it available to only the default Drupal site
       sites/example.com/themes
         making it available to only the example.com site if there is a
         sites/example.com/settings.php configuration file

  3. Log in as an administrator on your Drupal site and go to Administer > Site
     building > Themes (admin/build/themes) and make Alphorn or one of its
     sub-themes the default theme.

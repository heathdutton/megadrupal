ABOUT SQUID PRO
---------------

The Squid Pro theme is based on a 12-column 960 Grid System.

This Fluid-Width theme is built by Myndsets (http://www.myndsets.com) for Drupal 7.

Tested in Firefox 24.0, Google Chrome 30.0, Opera 17.0, Safari 5.1.7, IE7+, the Squid Pro theme is HTML5 and CSS3 compliant.

DEFAULT REGIONS
---------------

The Squid Pro theme is configured with 14 custom regions mentioned below:

REQUIREMENTS
------------

* 'jQuery Update' module (https://drupal.org/project/jquery_update)
* 'Libraries API' module (https://drupal.org/project/libraries)
* 'Superfish' library    (https://github.com/mehrpadin/Superfish-for-Drupal/archive/1.x.zip)
* 'jQuery Cycle' plugin  (http://malsup.github.io/jquery.cycle.all.js)
* 'jQuery Easing' plugin (https://github.com/gdsmith/jquery.easing/blob/master/jquery.easing.1.3.min.js)
* 'Font Awesome' library (http://fortawesome.github.io/Font-Awesome/3.2.1/)
* 'html5shiv' plugin (https://code.google.com/p/solarcdn/downloads/detail?name=html5shiv.min.js)

INSTALLATION
-------------
* Install as usual, see https://drupal.org/documentation/install/modules-themes for further information.

* jQuery Update module
  - Install the module
  - Configure jQuery version to v1.7 or later (admin/config/development/jquery_update)
  
* Libraries module
  - Install the module

* Superfish library
  - Download the zip file and extract it to sites/all/libraries/
  - PATH: sites/all/libraries/superfish/superfish.js

* jQuery Cycle plugin
  - Download the plugin and upload to sites/all/libraries/jquery.cycle
  - PATH: sites/all/libraries/jquery.cycle/jquery.cycle.all.js
     
* jQuery Easing plugin
  - Rename the file to jquery.easing.js and upload to sites/all/libraries/easing
  - PATH: sites/all/libraries/easing/jquery.easing.js
     
* Font Awesome library (RECOMMENDED version - 3.2.1)
  - Download the Font Awesome library and extract it to sites/all/libraries/
  - PATH: sites/all/libraries/font-awesome/css/font-awesome.css
  
* html5shiv plugin
  - Download the html5shiv.min.js and upload to sites/all/libraries/html5shiv
  - PATH: sites/all/libraries/html5shiv/html5shiv.min.js

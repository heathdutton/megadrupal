About Azhagu
============
Azhagu means beauty in Tamil language. This is a single and two-column 
responsive theme, compatible for desktops, tablets, phones and other devices.

Features
========
1.Responsive Layout.
2.Mobile Friendly.
3.Configurable Social Block for Facebook, Twitter, Google Plus and Pintrest.
4.Sass Compass Support
5.1 Column and 2 Columns layout
6.3 Columns Footer.
7.Total of 9 Regions
8.Compatible and tested on Firefox, Chrome browsers and Safari

Browser compatibility:
=====================
The theme has been tested on following browsers. Firefox, Chrome browsers 
and Safari

Drupal compatibility:
=====================
This theme is compatible with Drupal 7.x.x

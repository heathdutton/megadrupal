GlossyBlue writen by Shivcharan Patil(Shiv) of http://itapplication.net
A fixed-width Drupal theme using the PHPTemplate engine
--------------------------------------------------------------------------------

To use this theme, enable it and make sure you do the followings:
1. Enable Site Name and Site Slogan
2. Enable Search Box (in theme configuration)
3. In admin/content/comment/settings, set default display to
   Flat list - expanded   and
   Disable comment subject field

INTRODUCTION
------------

PointNet RedSun is a responsive Zen 7.x-5.x subtheme with performance
and mobile layout in mind.



FEATURES
------------

 - HTML5
 - Responsive design
 - Single or multi-column layout
 - css3 effects
 - Optimized for performance
 - Some of the features of Zen 7.x-5.x, please see:
   https://www.drupal.org/project/zen



REQUIREMENTS
------------

PointNet Red Sun is a subtheme of Zen, it requires Zen 7.x-5.x latest version:
https://www.drupal.org/project/zen



RECOMMENDED MODULES
-------------------

If you like to have an installation like the demo of this theme,
please install and configure the following modules:
 -FlexSlider ( https://www.drupal.org/project/flexslider )
 -Sticky Navigation ( https://www.drupal.org/project/stickynav )
 -Back to Top ( https://www.drupal.org/project/back_to_top )



INSTALLATION
------------

PointNet Red Sun is a Zen sub-theme, you should download and install both
PointNet Red Sun and Zen 7.x-5.x:

Download both the latest version of PointNet Red Sun and Zen 7.x-5.x,
unpack the theme folders and place them in your site's 'sites/all/themes,
please see https://drupal.org/getting-started/install-contrib/themes.

You should now have the following directory:
sites/all/themes/zen
sites/all/themes/pointnet_red_sun
Log into your site and go to the 'Appearance' page (/admin/appearance)
both PointNet Red Sun and Zen should be listed,
click the 'Enable and set default' link for PointNet Red Sun



CONFIGURATION
-------------

Please go to /admin/appearance/settings/pointnet_red_sun and disable:
Main menu, Rebuild theme registry on every page ;
Now you can go on /admin/structure/block and assign the "Main menu" block
to thr region "Navigation".

Configure as you would normally configure a contributed Drupal theme.

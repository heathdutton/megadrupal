Usage
============
* See Omega 4 Installation
http://drupal.org/project/omega

* Install compass
http://compass-style.org/

* Add gems
sudo gem install susy --pre
sudo gem install compass-normalize
sudo gem install compass-rgbapng

* Auto-compile sass files
compass watch

* Install libraries
either via
    bower install
or downloading manually and placing them under:
    cloudy/libraries/

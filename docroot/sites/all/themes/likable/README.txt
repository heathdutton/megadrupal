Likable Clean Theme
--------------------------------------------------------------------------
Likable Clean Theme is a clean, multi-column Drupal 7 theme with many block 
regions and a slideshow ready for use.

Clean design
HTML5
Fixed width 1000px
Multiple columns - 1, 2, or 3
Multi-level drop-down menus
Many block regions
Support custom logo
Ready to use slideshow



This theme supports the <a href="http://drupal.org/project/superfish">Superfish</a> 
module. How to setup the Superfish menu:

1. Copy the <a href="http://drupal.org/project/superfish">Superfish</a> module 
   to sites/all/modules or sites/sitename/modules.
2. Copy the <a href="http://drupal.org/project/libraries">Libraries API</a> 
   module to sites/all/modules or sites/sitename/modules as you require.
3. Download the latest version of the <a href="https://github.com/mehrpadin/Superfish-for-Drupal">Superfish library</a>
   and extract it to sites/all/libraries or sites/sitename/libraries as you require. 
   The extracted folder must be named superfish.
4. Enable Superfish and Libraries API modules.
5. Go to Administer > Menus > Primary links > Add link. 
6. Check the 'Show as expanded' option for parent menu items only if you want main menu as drop down.

The slideshow is based on the NIVO Slider.

Configuring NIVO Slider:
1. Copy the <a href="http://drupal.org/project/nivo_slider" rel="nofollow">Nivo Slider</a> 
   module to sites/all/modules or sites/sitename/modules.
2. Download the latest version of the <a href="http://nivo.dev7studios.com/pricing/">Nivo Slider jQuery plugin</a>
   and extract it to sites/all/libraries or sites/sitename/libraries as you require. 
   The extracted folder must be named nivo-slider.
3. Enable Nivo Slider module.
4. Add slides to the slideshow /admin/structure/nivo-slider.
5. An administration interface to configure slideshow settings in 'admin/structure/nivo-slider/options'.
6. Add block Nivo Slider to region 'Slideshow' in 'admin/structure/block'.
7. Clear your cache and refresh the page.




This theme is compatible with Drupal 7.x.x

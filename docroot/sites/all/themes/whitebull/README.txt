This is a whitebull theme in drupal 7.

Using Superfish Menu
--------------------------------------------------

1. Go to Structure > Menus.
2. Then create your own menu or create links in "Main Menu".
3. Go to Structure > Blocks.
4. Move the "Main Menu" block or menu block that you had been created.
5. Go to Appearance > Settings.
6. Uncheck "Main Menu" in Toggle display.
7. Finish.

*note
---------------------------------------------------
When you create superfish menu,
you must notice how many menus you created,
because the number of menus that can be loaded is limited.

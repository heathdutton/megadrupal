WHERE TO START
--------------

Welcome to Redfire!


INSTALLATION
------------
 1.Furst install Drupal 7. 

 2. Download Redfire from http://drupal.org/project/redfire

 3. Unpack the downloaded file, take the folders and place them in your
    Drupal installation under one of the following locations:
      sites/all/themes
        
    Note: you will need to create the "themes" folder under "sites/all/"
    or "sites/default/".

 3. Go to drupal 7 dashboard and find Appearance and click on it.

 4. Find Redfire theme and click on 'set default' option and save configuration.
